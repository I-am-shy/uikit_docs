# 集成

- - -

## 1 准备环境

在开始集成 ZEGO Express SDK 前，请确保开发环境满足以下要求：

- CentOS 6.5+ 64位。
- Ubuntu 14.04+ 64位。
- Linux 上已安装 CMake
- 麦克风等支持音频功能的外部设备。

## 2 集成 SDK

### 2.1 新建项目（可选）

<details> 
    <summary>此步骤以如何创建新项目为例，如果是集成到已有项目，可忽略此步。 </summary>  

创建工作目录并使用 CMake 管理

```c++
CMakeLists.txt ------------ cmake工程文件
src
    | main.cpp --------------- demo源文件
```

以下给出的是一个最简单的CMake 工程

```c++
cmake_minimum_required(VERSION 3.5.2)
project (ZegoExpressSDKDemo)

add_compile_options(
  -std=c++11
)

aux_source_directory(./src SRC_LIST)

add_executable(ZegoExpressSDKDemo ${SRC_LIST})
```

</details>

### 2.2 导入 SDK

1. 下载 SDK

    请从 [下载 SDK 包|_blank](#5580) 下载 SDK。

2. 解压 SDK，并拷贝到项目的目录下

    SDK 包含 “include” 和 “lib” 两个目录，每个目录包含的文件说明如下。

```c++
include    --------------- 包含SDK头文件
lib        --------------- 包含SDK的.lib和.dll文件
   | x86_64   --------------- 64位版本
```

3. 在 CMakeLists.txt 中引入 SDK

```c++
cmake_minimum_required(VERSION 3.5.2)
project (ZegoExpressSDKDemo)

// 指定头文件目录
include_directories("./include")
// 指定库文件目录
link_directories("./lib/x86_64")
// 链接库文件
link_libraries(ZegoExpressEngine rt)

add_compile_options(
  -std=c++11
)

aux_source_directory(./src SRC_LIST)

add_executable(ZegoExpressSDKDemo ${SRC_LIST})
```
