# 集成 SDK

---

## 1 准备环境

在开始集成 ZEGO Express SDK 前，请确保开发环境满足以下要求：

- 准备一台可以连接到互联网的 Windows 或 macOS 计算机。
- 使用最新版本的 Chrome 浏览器。

<div class="mk-hint">

浏览器的兼容性请参考 [浏览器兼容性和已知问题\|_blank](#12047)。   
</div>


## 2 集成 SDK

### 2.1 （可选）新建项目

<details class="zg-primary">
    <summary>此步骤以如何创建新项目为例，如果是集成到已有项目，可忽略此步。</summary>

1. 创建一个文件夹作为基础音视频通话的项目文件夹，结构类似如下： 

```
├── index.html
├── js
│   └── index.js
└── css
   └── index.css
```

- “index.html” 文件：为音视频通话应用的前端页面文件。
- “js” 目录下的 “index.js” 文件：为实现音视频通话功能的 JavaScript 代码。
- “css” 目录下的 “index.css” 文件：用于设置页面的样式。

2. 将以下代码拷贝到 “index.html” 文件中。

```html
<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>Zego RTC Audio Call</title>
    <link rel="stylesheet" href="./css/index.css">
</head>
<body>
    <h1>
        Zego RTC Audio Call
    </h1>
    <h4>Local Audio</h4>
    <audio id="local-audio" autoplay muted playsinline controls></audio>
    <h4>Remote Audio</h4>
    <audio id="remote-audio" autoplay playsinline controls></audio>
    <script src="./js/index.js"></script>
</body>
</html>
```

3. 将以下代码拷贝到 “index.css” 文件中。

```css
*{
    font-family: sans-serif;
}
h1,h4{
    text-align: center;
}
audio {
    width: 300px;
    height: auto;
}
#local-audio{
    position: relative;
    margin: 0 auto;
    display: block;
}

#remote-audio{
    display: flex;
    height: auto;
    margin: auto;
    position: relative !important;
}
```


</details>


### 2.2 导入 SDK

开发者可通过以下任意一种方式集成 SDK。

#### 方式一：从官网下载 SDK，手动集成 

1. 下载最新版本的 [SDK\|_blank](#3209) 并解压。

2. 解压 SDK 压缩包，我们可以在 “dist_js/” 下找到 “ZegoExpressWebRTC-x.x.x.js” 文件。
3. 推荐下载 [VSCode](https://code.visualstudio.com/)，在 VSCode 中找到项目文件目录并打开。
<img src="/Pics/Express/VScode_open.jpeg" width="80%">
4. 安装 VSCode 扩展插件 Live Server。
<img src="/Pics/Express/VScode_liveserver.jpeg" width="80%">
5. 在 “index.html” 文件中编写界面代码，并在文件的 head 中引入 “ZegoExpressWebRTC-x.x.x.js”。其中，“x.x.x” 为 SDK 的版本号，请参考第 2 步压缩包解压后的文件名修改。

    ```html
    <script src="ZegoExpressWebRTC-x.x.x.js"></script>
    ``` 
6. 在 “index.html” 文件的编辑面板，单击鼠标右键，弹出菜单，选择“Open with Live Server”，即可看到基本界面，此时表示 SDK 已集成。

    <img src="/Pics/Express/VScode_open_liveserver.jpeg" width="70%">



#### 方式二： 使用 npm 获取 SDK

1. 执行 `npm i zego-express-engine-webrtc` 命令安装依赖。

<div class="mk-hint">

- npm 下载包支持 typescript 语言（推荐）。
- 如果在 macOS 或 Linux 系统中执行 npm 命令失败，提示 “permission denied”，请在 npm 命令前加上 `sudo` 重新执行即可。

</div>

2. 在 “index.js” 文件中引入 SDK。

```javascript
import {ZegoExpressEngine} from 'zego-express-engine-webrtc'
```
或

```javascript
var ZegoExpressEngine = require('zego-express-engine-webrtc').ZegoExpressEngine
```
3. 在本地 Web 服务器上运行和测试该项目。

本地服务器（localhost 或 127.0.0.1）运行 Web 应用仅作为测试，生产环境必须使用 https 协议。

  a. 在命令行中执行 `npm i live-server -g` 命令，安装 live-server。

  b. 进入项目所在的根目录。

  c. 执行 `live-server .` 命令运行该项目，浏览器会自动打开 “index.html” 页面。
