- - -

@@@Express_collection_rotate_info@@@

## 实现流程

<div class="mk-warning">

以下实现流程仅适用于固定比例模式、自适应模式及对齐模式，如需自定义视频采集旋转方式，请参考 [如何自定义视频采集旋转方式？](http://doc-zh.zego.im/faq/express_video_capture_rotation?product=ExpressVideo&platform=android)

</div>

### 推流端

推流端需要调用 [setAppOrientationMode ](@setAppOrientationMode) 接口设置任意一种采集旋转模式。

<div class="mk-warning">

请在调用 [createEngine](@createEngine) 后立刻调用 [setAppOrientationMode](@setAppOrientationMode) 接口，再调用 [startPublishingStream](@startPublishingStream) 或 [startPreview](@startPreview) 接口。
 
</div>


<div class="multiple-select-codes">
    <div class="code-tabs hide-scrollbar">
      <div class="scroll-box">
        <span class="tab-item">
          <span>设置为固定比例模式</span>
        </span>
        <span class="tab-item">
          <span>设置为自适应模式</span>
        </span>
        <span class="tab-item">
          <span>设置为对齐模式</span>
        </span>
      </div>
    </div>

  <div class="code-list">

ZegoExpressEngine.getEngine().setAppOrientationMode(ZegoOrientationMode.FIXED_RESOLUTION_RATIO);

  ```java
ZegoExpressEngine.getEngine().setAppOrientationMode(ZegoOrientationMode.ADAPTION);
  ```
  
  ```java
ZegoExpressEngine.getEngine().setAppOrientationMode(ZegoOrientationMode.ALIGNMENT);
  ```
  </div>
</div>

### 拉流端

**仅自适应模式下**，拉流端需要在拉流前调用 [setAppOrientationMode](@setAppOrientationMode) 接口设置自适应模式。

<div class="mk-warning">


- 请在调用 [createEngine](@createEngine) 后，立刻调用 [setAppOrientationMode](@setAppOrientationMode) 接口设置视频的朝向模式，再调用 [startPlayingStream](@startPlayingStream) 开始拉流。 
- 自适应模式下，仅支持从 RTC 拉流。
</div>

```java
ZegoExpressEngine.getEngine().setAppOrientationMode(ZegoOrientationMode.ADAPTION);
```
