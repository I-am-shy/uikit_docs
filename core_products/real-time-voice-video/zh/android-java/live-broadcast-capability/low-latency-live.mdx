## 使用步骤

设置 [ZegoPlayerConfig\|_blank](/zh/api?doc=Express_Video_SDK_API~Java_android~class~im-zego-zegoexpress-entity-zego-player-config#resource-mode) 的 “resourceMode” 参数为 “ONLY_L3”，调用 [startPlayingStream\|_blank](https://doc-zh.zego.im/article/api?doc=Express_Video_SDK_API~Java_android~class~im-zego-zegoexpress-zego-express-engine&jumpType=route#start-playing-stream-1) 接口，根据传入的流 ID 参数 “streamID”，拉取远端推送的低延迟直播流。

```java
ZegoPlayerConfig playerConfig = new ZegoPlayerConfig();
playerConfig.resourceMode = ZegoStreamResourceMode.ONLY_L3;
ZegoExpressEngine.getEngine().startPlayingStream(streamID, playCanvas, playerConfig);
```
