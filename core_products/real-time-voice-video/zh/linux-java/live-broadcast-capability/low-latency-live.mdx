## 2 前提条件

在使用超低延迟直播前，请确保已在项目中实现基本的实时音视频功能，详情请参考 [快速开始 - 集成](#7680) 相关文档。

## 3 使用步骤

设置 [ZegoPlayerConfig](/zh/api?doc=Express_Video_SDK_API~Java_linux~class~im-zego-zegoexpress-entity-zego-player-config) 的 “resourceMode” 参数为 “ONLY_L3”，调用 [startPlayingStream](/zh/api?doc=Express_Video_SDK_API~Java_linux~class~im-zego-zegoexpress-zego-express-engine#start-playing-stream) 接口，根据传入的流 ID 参数 “streamID”，拉取远端推送的超低延迟直播流。

```java
ZegoPlayerConfig playerConfig = new ZegoPlayerConfig();
playerConfig.resourceMode = ZegoStreamResourceMode.ONLY_L3;
ZegoExpressEngine.getEngine().startPlayingStream(streamID, playCanvas, playerConfig);
```
