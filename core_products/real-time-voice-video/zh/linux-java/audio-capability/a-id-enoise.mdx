## 使用步骤

@@@AIDenoise_mode@@@


```java
// 开启 ANS
engine.enableANS(true);
// 注意：设置 ANS 模式 ZegoANSMode 后，ZEGO Express SDK 会强制关闭瞬态噪声抑制 [enableTransientANS]
engine.setANSMode(ZegoANSMode.AI);
// 可选：设置为 AI_BALANCED 模式
engine.setANSMode(ZegoANSMode.AI_BALANCED);
```
