# 集成 SDK

---

## 准备环境

在开始集成 ZEGO Express SDK 前，请确保开发环境满足以下要求：

- 任意具有 GLIBC 2.16 或以上版本的 Linux 发行版系统，支持 x86_64, aarch64, armhf, armel 架构
- libasound（ALSA）。
- libv4l2（v4l utils）。
- JDK 1.7 或以上版本。
- Apache Maven 3.6 或以上版本。

<Note title="说明">


- SDK 依赖 libasound (ALSA) 和 libv4l2 (v4l utils)。
- CentOS (RHEL/Fedora) 可以通过执行 `yum install alsa-lib-devel libv4l-devel` 命令安装。
- Ubuntu (Debian/Deepin) 可以通过执行 `apt install libasound2-dev libv4l-dev` 命令安装。
- 其他平台和系统请自行安装。
- 若需要交叉编译，请参考 [如何交叉编译 Linux alsa-lib 依赖库？](http://doc-zh.zego.im/faq/alsa_lib_cross_compile?product=ExpressAudio&platform=linux) 和 [如何交叉编译 Linux v4l-utils 依赖库？](http://doc-zh.zego.im/faq/v4l_utils_cross_compile?product=ExpressVideo&platform=linux) 两篇文档，同时目标机器需安装好 libasound 和 libv4l2 依赖库。


</Note>



## 集成 SDK

请通过 [下载](https://doc-zh.zego.im/article/5569) 文档，下载最新版本的 SDK。

<Note title="说明">


建议参考 [跑通示例源码](https://doc-zh.zego.im/article/14512) 的工程来集成 SDK。

</Note>







### 方式 1：安装 jar 到本地 maven

```sh
$ mvn install:install-file -Dfile=ZegoExpressEngine.jar -DgroupId=im.zego.express -DartifactId=ZegoExpressEngine -Dversion=1 -Dpackaging=jar`
```

然后修改您工程的 “pom.xml” 配置文件，在 `<dependencies>` 中添加依赖：

```xml
  <dependencies>
    <dependency>
      <groupId>im.zego.express</groupId>
      <artifactId>ZegoExpressEngine</artifactId>
      <version>1</version>
    </dependency>
  </dependencies>
```

### 方式 2：通过 IDE 导入 jar 到工程

1. 打开 IntelliJ IDEA 并打开您的 Java 工程。

2. 添加 SDK 引用到工程中。

    a. 打开 IDEA，在菜单栏选择 “File > Project Structure...”。
    <Frame width="512" height="auto" caption=""><img src="https://storage.zego.im/sdk-doc/Pics/Linux/Express/idea_open_project_structure.png" /></Frame>

    b. 在弹出的 “Project Structure” 对话框中，选择菜单 “Libraries”，单击 “+”，选择 “Java”。
    <Frame width="512" height="auto" caption=""><img src="https://storage.zego.im/sdk-doc/Pics/Linux/Express/idea_add_jar_to_project.png" /></Frame>

    c. 选择 SDK 的 jar 包后，单击“Open”导入。
    <Frame width="512" height="auto" caption=""><img src="https://storage.zego.im/sdk-doc/Pics/Linux/Express/idea_open_express_jar.png" /></Frame>

### 注意事项

集成 SDK 后，运行 SDK 时需要依赖 Native 动态库，请参考 [跑通示例源码](https://doc-zh.zego.im/article/14512) 的运行指引。
