## 使用步骤

设置 [ZegoWebPlayOption](@-ZegoWebPlayOption) 的 “resourceMode” 参数为 “2” 后，调用 [startPlayingStream](@startPlayingStream) 接口，根据传入的流 ID 参数 “streamID”，拉取远端推送的超低延迟直播流。

```javascript
const playOption = {resourceMode: 2};
const remoteStream = await zg.startPlayingStream(streamID, playOption)
```
