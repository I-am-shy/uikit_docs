### 3 拉转码流

配置 [ZegoWebPlayOption > codecTemplateID\|_blank](/article/api?doc=Express_Video_SDK_API~javascript_web~interface~ZegoWebPlayOption#codec-template-id) 属性，填写 [步骤 1 添加转码配置模板](!Single_stream_transcoding#step1) 中获取的转码模板 ID，并开始拉流。

<div class="mk-warning">

CDN 拉流也必须使用 [startPlayingStream](@startPlayingStream) 方法，不能直接使用 URL 拉流，不然无法触发单流转码。
</div>

```js
const stream = await zg.startPlayingStream(streamID, {
    codecTemplateID: 360, // 转码模板 ID
}
```
