### 2 设置地理围栏

@@@Express_Geofencing_step2@@@

在创建 SDK 之前，调用 [setGeoFence](@setGeoFence) 接口，设置地理围栏信息。

<div class="mk-warning">

请在创建 [ZegoExpressEngine](@-ZegoExpressEngine) 引擎实例之前配置地理围栏信息，否则无效。
</div>


```java
// 设置联合模式的示例
const geofenceType = 1;//设置联合模式
const geofenceAreaList = [2,4]; // 设置区域列表信息，最少设置 1 个，最多设置不得大于 SDK 所支持个数
ZegoExpressEngine.setGeoFence(geofenceType, geofenceAreaList);// 这个接口是静态方法，在 new ZegoExpressEngine 之前调用

// 设置排除模式的示例
const geofenceType = 2;//设置排除模式
const geofenceAreaList = [2,4];// 设置区域列表信息，最少设置 1 个，最多设置不得大于等于 SDK 所支持个数
ZegoExpressEngine.setGeoFence(geofenceType, geofenceAreaList);// 这个接口是静态方法，在 new ZegoExpressEngine 之前调用
```
