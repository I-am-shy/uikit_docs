# 场景化 AI 降噪

- - -

@@@Expres_AI_Denoise2@@@

## 前提条件

在实现 AI 降噪功能之前，请确保：

- 已在 [ZEGO 控制台](https://console.zego.im) 创建项目，并申请有效的 AppID 和 AppSign，详情请参考 [控制台 - 项目信息](https://doc-zh.zego.im/article/12107)。
- 已在项目中集成 ZEGO Express SDK，并实现了基本的音视频推拉流功能，详情请参考 [快速开始 - 集成](!Integration/SDK_Integration) 和 [快速开始 - 实现流程](!Integration/Solution_Implementation)。


## 使用步骤

@@@AIDenoise_mode@@@

4. 开启噪声抑制后，开发者可通过调用 [setANSMode](@setANSMode) 接口，设置 ANS 模式为 “ZegoANSMode.AI” 模式，开启 AI 降噪功能。



```ts
// 开启 ANS
ZegoExpressEngine.instance.enableANS(true)
// 注意：设置 ANS 模式为 ZegoANSMode.AI 后，ZEGO Express SDK 会强制关闭瞬态噪声抑制 [enableTransientANS]
ZegoExpressEngine.instance.setANSMode(ZegoANSMode.AI)
```
