## 使用步骤

@@@AIDenoise_mode@@@

4. 开启噪声抑制后，开发者可通过调用 [setANSMode\|_blank](@setANSMode) 接口，设置 ANS 模式为 “ZegoANSMode.AI” 模式，开启 AI 降噪功能。



```ts
// 开启 ANS
ZegoExpressEngine.instance.enableANS(true)
// 注意：设置 ANS 模式为 ZegoANSMode.AI 后，ZEGO Express SDK 会强制关闭瞬态噪声抑制 [enableTransientANS]
ZegoExpressEngine.instance.setANSMode(ZegoANSMode.AI)
```
