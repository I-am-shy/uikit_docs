# 查询媒体文件任务

- - -

## 1 描述

本接口用于查询 [合并媒体文件](!Media_File/MergeMedia)、[开始点播转码](!Start_onDemand_transcoding) 及 [开始点播截图](!MediaThumbnails) 任务的详细信息，作为 [媒体文件合并完成回调](!Callback/compose) 和 [点播转码完成回调](!Callback_onDemand_transcoding) 的补充。

本接口支持查询“合并任务”、“转码任务”及“截图任务”的进度；任务完成后，也可以通过本接口获取合并文件或转码文件的播放地址。

## 2 接口原型

- 请求方法：GET
- 请求地址：https://rtc-api.zego.im/?Action=DescribeMediaTask
- 传输协议：HTTPS
- 调用频率限制：20 次/秒
