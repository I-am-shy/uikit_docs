# 停止 CDN 录制

---

## 1 描述

调用本接口停止 CDN 录制。相关回调请参考 [录制文件生成回调\|_blank](!Callback/Record_File_Create_Callback)。



## 2 接口原型

- 请求方法：GET
- 请求地址：https://rtc-api.zego.im/?Action=StopCDNRecord
- 传输协议：HTTPS
- 调用频率限制：20 次/秒

## 4 请求示例

```
https://rtc-api.zego.im/?Action=StopCDNRecord
&StreamId[]=cdn01
&StreamId[]=cdn02
&Vendor=Tencent
&<公共请求参数>
```
