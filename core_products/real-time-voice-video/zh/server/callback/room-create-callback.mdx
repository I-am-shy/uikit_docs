# 房间创建回调

- - -

## 1 描述

当开发者需要了解某个房间创建的相关信息时，可以通过本回调获取。

@@@Server_API2_callback_intro@@@

<tr>

## 4 数据示例

```json
{
    "event": "room_create",
    "appid": 1,
    "timestamp": 1499676978,
    "nonce": "350176",
    "signature": "signature",
    "room_id": "rid_1242649",
    "room_session_id": 858012925204410400,
    "room_create_time": "1499676978027",
    "id_name": "id123"
}
```

## 5 返回响应

返回 HTTP status code 为 2XX （例如 200）表示成功，其他响应都表示失败。
