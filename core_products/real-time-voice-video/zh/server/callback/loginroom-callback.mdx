# 登录房间回调

- - -


## 1 描述

当开发者需要了解目前房间内用户的相关信息时，可以通过接收 “房间登录回调” 信息，并结合 [退出房间回调\](!Callback/Logoutroom_callback) 进行分析。

## 5 返回响应


返回 HTTP status code 为 2XX （例如 200）表示成功，其他响应都表示失败。
