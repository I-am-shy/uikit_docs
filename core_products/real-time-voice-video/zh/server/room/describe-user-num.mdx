# 获取房间人数

---

## 1 描述

调用本接口获取房间人数。


## 2 接口原型

- 请求方法：GET
- 请求地址：https://rtc-api.zego.im/?Action=DescribeUserNum
- 传输协议：HTTPS
- 调用频率限制（同一个 AppID 下所有房间）：10 次/秒（测试环境：1 次/秒）
