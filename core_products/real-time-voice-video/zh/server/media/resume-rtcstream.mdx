# 恢复 RTC 推流

---

## 1 描述

调用本接口恢复 RTC 推流，可恢复被禁止推往 RTC 媒体服务的流 ID 。

## 2 接口原型

- 请求方法：GET
- 请求地址：https://rtc-api.zego.im/?Action=ResumeRTCStream
- 传输协议：HTTPS
- 调用频率限制：20 次/秒（测试环境：5 次/秒）
