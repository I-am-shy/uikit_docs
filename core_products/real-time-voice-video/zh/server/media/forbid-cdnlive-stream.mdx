# 禁止 CDN 推流

---

## 1 描述

调用本接口禁止 CDN 推流，适用直推 CDN 场景。如需应用于转推 CDN 场景，请使用 [停止旁路推流](!media/DeleteCDNTransferRule) 接口。


<div class="mk-warning">

首次使用本接口，需要联系 ZEGO 技术支持进行配置。
</div>

## 2 接口原型

- 请求方法：GET
- 请求地址：https://rtc-api.zego.im/?Action=ForbidCDNLiveStream
- 传输协议：HTTPS
- 调用频率限制：20 次/秒
