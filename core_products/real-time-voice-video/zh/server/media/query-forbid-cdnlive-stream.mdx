# 查询被禁止 CDN 推流的列表

- - -
<style>
table th:nth-of-type(1) {width: 30%;}
table th:nth-of-type(2) {width: 30%;}
table {width: 100%;}
</style>

## 1 描述

调用本接口查询已被禁止推流到 CDN 的 URL 列表。


<div class="mk-warning">

首次使用本接口，需要联系 ZEGO 技术支持进行配置。
</div>

## 2 接口原型

- 请求方法：GET
- 请求地址：https://rtc-api.zego.im/?Action=QueryForbidCDNLiveStream
- 传输协议：HTTPS
- 调用频率限制：20 次/秒
