# 视频采集旋转

- - -

## 功能简介

@@@Express_collection_rotate_info@@@

## 实现流程

<div class="mk-warning">

以下实现流程仅适用于固定比例模式、自适应模式及对齐模式，如需自定义视频采集旋转方式，请参考 [如何自定义视频采集旋转方式？\|_blank](http://doc-zh.zego.im/faq/express_video_capture_rotation?product=ExpressVideo&platform=ios)
</div>

### 推流端

推流端需要调用 [setAppOrientationMode\|_blank](@setAppOrientationMode) 接口设置任意一种采集旋转模式。

<div class="mk-warning">

请在调用 [createEngineWithProfile\|_blank](@createEngineWithProfile) 后立刻调用 [setAppOrientationMode\|_blank](@setAppOrientationMode) 接口，再调用 [startPublishingStream\|_blank](@startPublishingStream) 或 [startPreview\|_blank](@startPreview) 接口。
 
</div>

<div class="multiple-select-codes">
    <div class="code-tabs hide-scrollbar">
      <div class="scroll-box">
        <span class="tab-item">
          <span>设置为固定比例模式</span>
        </span>
        <span class="tab-item">
          <span>设置为自适应模式</span>
        </span>
        <span class="tab-item">
          <span>设置为对齐模式</span>
        </span>
      </div>
    </div>

  <div class="code-list">
  
  ```objc
[[ZegoExpressEngine sharedEngine]setAppOrientationMode:ZegoOrientationModeFixedResolutionRatio];
  ```
  
  ```objc
[[ZegoExpressEngine sharedEngine]setAppOrientationMode:ZegoOrientationModeAdaption];
  ```
  
  ```objc
[[ZegoExpressEngine sharedEngine]setAppOrientationMode:ZegoOrientationModeAlignment];
  ```
  </div>
</div>

### 拉流端

**仅自适应模式下**，拉流端需要在拉流前调用 [setAppOrientationMode\|_blank](@setAppOrientationMode) 接口设置自适应模式。

<div class="mk-warning">


- 请在调用 [createEngineWithProfile\|_blank](@createEngineWithProfile) 后，立刻调用 [setAppOrientationMode\|_blank](@setAppOrientationMode) 接口设置视频的朝向模式，再调用 [startPlayingStream\|_blank](@startPlayingStream) 开始拉流。 
- 自适应模式下，仅支持从 RTC 拉流。
</div>



```objc
[[ZegoExpressEngine sharedEngine]setAppOrientationMode:ZegoOrientationModeAdaption];
```
