## 使用步骤

设置 [ZegoPlayerConfig\|_blank](/zh/api?doc=Express_Video_SDK_API~CPP_windows~struct~zego-express-zego-player-config) 的 “resourceMode” 参数为 “ZEGO_STREAM_RESOURCE_MODE_ONLY_L3”，调用 [startPlayingStream\|_blank](https://doc-zh.zego.im/article/api?doc=Express_Video_SDK_API~cpp_windows~class~IZegoExpressEngine#start-playing-stream-1) 接口，根据传入的流 ID 参数 “streamID”，拉取远端推送的低延迟直播流，即可实现超低延迟直播功能。

```c++
ZegoPlayerConfig playerConfig;
playerConfig.resourceMode = ZEGO_STREAM_RESOURCE_MODE_ONLY_L3;
engine->startPlayingStream(streamID, &playCanvas, playerConfig);
```
