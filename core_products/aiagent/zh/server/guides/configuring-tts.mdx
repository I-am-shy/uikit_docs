# 配置 TTS

## 功能简介

为匹配不同的人设和场景，您可能需要：
- 选择不同的文本转语音(TTS)厂商，例如火山引擎、MiniMax、阿里云等，
- 配置不同的音色。
- 针对 TTS 的音频做个性化调整，例如音量、语速、语调等；
- 特殊规则过滤 TTS 的内容。例如“（开心地说）今天天气真好呀”，过滤括号内容。

## 前提条件

1. 开通 AI Agent 服务
2. 开通对应 TTS 厂商服务：
    - 方式1:直接通过 zego_test 账号体验
    - 方式2:通过 ZEGO 购买 TTS 服务。请联系 ZEGO 商务获取账号及鉴权信息
    - 方式3：自行购买 TTS 服务，获取密钥信息等

## TTS 参数说明 <a id="tts-params" />


| 参数名         | 类型            | 是否必填 | 说明                                                                                                                                                                                                                                                                                                                                                   |
| -------------- | --------------- | -------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Vendor         | String          | 是       | 文本转语音（TTS）服务提供商。可选值：<ul><li>Aliyun: 阿里云 TTS（注意：这是普通语音合成，不是 CosyVoice ）。</li><li>CosyVoice: 阿里云 CosyVoice TTS</li><li>ByteDance: 火山引擎单向流式 TTS。</li><li>ByteDanceFlowing: 火山双向流式引擎 TTS。</li><li>MiniMax: MiniMax TTS</li></ul> <br/><Note title="说明">此参数在更新智能体实例时无法更新</Note> |
| Params         | Object          | 是       | TTS 配置参数，格式为 JSON 对象。包含 app 参数（用于认证）和其他参数（用于调整 TTS 效果）。详见下方 Params 参数说明。                                                                                                                                                                                                                                   |
| FilterText     | Array of Object | 否       | 过滤指定标点符号中的文字后，再进行语音合成。例如：要过滤 [] 中的内容，则设置成`[{"BeginCharacters": "[", "EndCharacters": "]"}]`<br/><Note title="说明">此参数在更新智能体实例时无法更新</Note>                                                                                                                                                        |
| TerminatorText | String          | 否       | 可用于设置 TTS 的终止文本。若输入 TTS 的文本中出现匹配 TerminatorText 字符串的内容，则本轮 TTS 从 TerminatorText 字符串（包含）开始的内容将不再进行语音合成。<br/><Note title="说明">双向流式只能设置一个字符。最大长度：4 个字符。</Note>                                                                                                             |
