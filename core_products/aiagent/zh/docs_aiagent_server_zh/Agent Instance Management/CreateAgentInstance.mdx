# 创建智能体实例

---

## 描述

通过向本接口，您可以创建智能体实例，并将智能体实例加入到文本对话和语音（RTC）对话之中。

## 接口原型

- 请求方法：POST
- 请求地址：`https://aigc-chat-api.zegotech.cn?Action=CreateAgentInstance`
- 传输协议：HTTPS
- 调用频率限制：10 次/秒

## 请求参数

以下请求参数列表仅列出了接口请求参数和部分公共参数，完整公共参数列表请参考 [调用方式 - 公共请求参数](./../Accessing%20Server%20APIs.mdx#公共请求参数)。

| 参数            | 类型    | 是否必选 | 描述                                                         |
|-----------------|---------|----------|--------------------------------------------------------------|
| AgentId         | String  | 是       | 智能体的唯一标识。仅支持数字、英文字符、'-'、'_'、'@'，长度不得超过 32 字节。|
| UserId          | String  | 是       | 用于登录 RTC 房间的真实用户 ID。 仅支持数字、英文字符、'-'、'_'，长度不得超过 32 字节。                   |
| RtcInfo         | Object  | 是       | RTC 相关信息。本参数结构，请参考 [RtcInfo](#rtcinfo)。                                           |
| LLM             | Object  | 否       | 大语言模型参数。本参数结构，请见 [常用参数说明 - LLM](./../Common%20Parameter%20Description.mdx#llm)。                                               |
| TTS             | Object  | 否       | 文本转语音参数。本参数结构，请见 [常用参数说明 - TTS](./../Common%20Parameter%20Description.mdx#tts)。                                               |
| ASR             | Object  | 否       | 语音识别参数。本参数结构，请见 [常用参数说明 - ASR](./../Common%20Parameter%20Description.mdx#asr)。                                               |
| MessageHistory   | Object  | 否       | 供智能体实例使用的历史消息，最多 100 条。本参数结构，请参考 [MessageHistory](#messagehistory)。                           |

### RtcInfo

<Note title="说明">

RoomId、AgentStreamId、AgentUserId、UserStreamId 说明如下：
- 字符限制：仅支持数字、英文字符、‘-’、‘_’。
- 长度限制：
    - RoomId、AgentStreamId、UserStreamId：128 字节。
    - AgentUserId：32 字节。
</Note>

| 参数           | 类型    | 是否必选 | 描述                                                         |
|----------------|---------|----------|--------------------------------------------------------------|
| RoomId         | String  | 是       | RTC 房间 ID。                                             |
| AgentStreamId  | String  | 是       | 智能体实例推流使用的流 ID。                                                      |
| AgentUserId    | String  | 是       | 智能体实例的用户 ID。                                          |
| UserStreamId   | String  | 是       | 真实用户推流使用的流 ID。                               |
| WelcomeMessage  | String  | 否       | 语音对话欢迎语。如果配置此参数，每次用户进入对话时，智能体实例都会发送此内容。若此参数为空，则不发送欢迎语。     |

### MessageHistory

| 参数          | 类型      | 是否必选 | 描述                                                                                     |
|---------------|-----------|----------|------------------------------------------------------------------------------------------|
| SyncMode      | Number       | 否       | 消息同步模式：<ul><li>0：从 ZIM 同步。<Note title="说明"><ul><li>若该 UserID 未通过 ZIM 客户端登录服务或未在 ZIM 服务端完成注册，实时互动 AI Agent 后台将自动为其执行 ZIM 服务注册流程。</li><li>建议您提前注册该用户，以便完善用户信息设置并提升智能体实例的创建效率。</li></ul></Note></li><li>1：通过下列 `Messages` 参数同步。</li></ul> |
| Messages      | Array of Object  | 否       | 消息列表。本参数结构，请参考 [Messages](#messages)。                                                                                          |
| WindowSize    | Number       | 否       | 以最近多少条历史消息作为上下文供 LLM 使用。默认 20 条，最大 100 条。取值范围为 [0, 100]。                                                    |
| ZimRobotId    | String    | 否  | ZIM 机器人 ID，用于加载用户与该 ZIM 机器人的聊天上下文，并将对话过程中产生的消息同步至 ZIM。如果此参数为空，实时互动 AI Agent 后台将随机生成。<Note title="说明">仅当 `SyncMode` 为 0 时有效。</Note> |

### Messages

| 参数     | 类型   | 是否必选 | 描述                                                                 |
|----------|--------|----------|----------------------------------------------------------------------|
| Role     | String | 是       | 消息发送者角色。取值如下：<ul><li>`user`：用户。</li><li>`assistant`：智能体</li></ul> |
| Content  | String | 是       | 发送的内容。                                                        |

## 请求示例

- 请求地址 URL：

    ```https
    https://aigc-chat-api.zegotech.cn?Action=CreateAgentInstance
    &<公共请求参数>
    ```

- 请求消息体：

    ```json
    {
        "AgentId": "zy-126",
        "UserId": "u123",
        "TTS": {
            "Vendor": "Bytedance",
            "Params": {
                "app": {
                    "appid": "xxxxxxxxxxx",
                    "token": "xxxxxxxxxxx",
                    "cluster": "volcano_tts"
                },
                "speed_ratio": 1,
                "volume_ratio": 1,
                "pitch_ratio": 1,
                "emotion": "happy",
                "audio": {
                    "rate": 24000,
                    "voice_type1": "zh_female_wenroushunv_mars_bigtts",
                    "voice_type": "BV700_streaming"
                }
            }
        },
        "RtcInfo": {
            "RoomId": "124",
            "AgentStreamId": "a124",
            "AgentUserId": "a124",
            "UserStreamId": "u124",
            "WelcomeMessage": ""
        },
        "MessageHistory": {
            "SyncMode": 1,
            "Messages": [],
            "WindowSize": 10
        }
    }
    ```

## 响应参数

| 参数 | 类型 | 描述 |
|------|------|------|
| Code | Number | 返回码。0 表示成功，其他值表示失败。如需了解错误码及响应处理建议，请参考 [返回码](./../Return%20codes.mdx)。 |
| Message | String | 请求结果的说明信息。 |
| RequestId | String | 请求 ID。 |
| Data | Object | 返回数据。 |
| └AgentInstanceId | String | 智能体实例的唯一标识。 |
