# 注册智能体

---

##  描述

通过向本接口传入基础配置，您可以注册一个智能体（Agent），以用于创建智能体实例。

## 接口原型

- 请求方法：POST
- 请求地址：`https://aigc-chat-api.zegotech.cn?Action=RegisterAgent`
- 传输协议：HTTPS
- 调用频率限制：10 次/秒

## 请求参数

<Note title="说明">
下列 ID 类参数，包括 ConversationId、UserId、AgentId、AgentTemplateId 等，应当遵循以下规则：
- AgentId 的字符限制：数字、英文字符、'-'、'_'、'@'。
- 其它 ID 类参数的字符限制：数字、英文字符、'-'、'_'。
- 长度上限：
  - ConversationId、AgentTemplateId：128 字节。
  - UserId、AgentId：32 字节。
</Note>

| 参数               | 类型   | 是否必选 | 描述                                                                                                                                   |
|------------------|------|------|--------------------------------------------------------------------------------------------------------------------------------------|
| AgentId          | String | 是    | 智能体的唯一标识。<Note title="说明">请保证 AgentId 在同一 AppID 下唯一。</Note>                                                              |
| AgentConfig | Object | 否    | 自定义智能体参数，本参数结构，请见 [常用参数说明 - CustomAgentConfig](./../Common%20Parameter%20Description.mdx#customagentconfig)。若 `AgentTemplateId` 不为空，则在该模版基础上，根据本参数所传内容做修改。                  |

## 请求示例

- 请求地址 URL：

```https
https://aigc-chat-api.zegotech.cn?Action=RegisterAgent
&<公共请求参数>
```

- 请求消息体：
```json
{
    "AgentId": "xiaozhi",
    "AgentConfig": {
        "Name": "小智",
        "LLM": {
            "Url": "https://ark.cn-beijing.volces.com/api/v3/chat/completions",
            "ApiKey": "eexxxxxxxxxxxxxxx",
            "Model": "ep-xxxxxxxxxx",
            "SystemPrompt": "你是一个问答机器人"
        },
        "TTS": {
            "Vendor": "Bytedance",
            "Params": {
                "app": {
                    "appid": "123456789",
                    "token": "xxxxxxxxxxxxxxx",
                    "cluster": "volcano_tts"
                },
                "speed_ratio": 1,
                "volume_ratio": 1,
                "pitch_ratio": 1,
                "emotion": "happy",
                "audio": {
                    "rate": 24000,
                    "voice_type": "zh_female_wanwanxiaohe_moon_bigtts"
                }
            }
        },
        "ASR": {}
    }
}
```

## 响应参数

| 参数 | 类型 | 描述 |
|------|------|------|
| Code | Number | 返回码。0 表示成功，其他值表示失败。如需了解错误码及响应处理建议，请参考 [返回码](./../Return%20codes.mdx)。 |
| Message | String | 请求结果的说明信息。 |
| RequestId | String | 请求 ID。 |

## 响应示例

```json
{
    "Code": 0,
    "Message": "success",
    "RequestId": "1898302332764950528"
}
```
