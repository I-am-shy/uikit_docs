# 删除智能体

---

##  描述

通过向本接口传入智能体 ID，您可以删除一个智能体。

## 接口原型

- 请求方法：POST
- 请求地址：`https://aigc-chat-api.zegotech.cn?Action=DeleteAgent`
- 传输协议：HTTPS
- 调用频率限制：10 次/秒

## 请求参数

以下请求参数列表仅列出了接口请求参数和部分公共参数，完整公共参数列表请参考 [调用方式 - 公共请求参数](./../Accessing%20Server%20APIs.mdx#公共请求参数)。

| 参数               | 类型   | 是否必选 | 描述             |
|------------------|------|------|-------------------------|
| AgentId          | String | 是    | 智能体的唯一标识。仅支持数字、英文字符、'-'、'_'、'@'，长度不得超过 32 字节。<Note title="说明">请保证 AgentId 在同一 AppID 下唯一。</Note>                                                              |

## 请求示例

- 请求地址 URL：

    ```https
    https://aigc-chat-api.zegotech.cn?Action=DeleteAgent
    &<公共请求参数>
    ```

- 请求消息体：
    ```json
    {
        "AgentId": "zy-123"
    }
    ```

## 响应参数

| 参数 | 类型 | 描述 |
|------|------|------|
| Code | Number | 返回码。0 表示成功，其他值表示失败。如需了解错误码及响应处理建议，请参考 [返回码](./../Return%20codes.mdx)。 |
| Message | String | 请求结果的说明信息。 |
| RequestId | String | 请求 ID。 |
| Data | Object | 此请求结果为 `null`。 |

## 响应示例

```json
{
    "Code": 0,
    "Message": "Succeed",
    "RequestId": "1843985617336143872",
    "Data": null
}
```
