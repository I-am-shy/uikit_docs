# 查询模版
---

##  描述

本接口可用于获取可用的智能体模板列表，或者根据智能体模板 ID 查询详细信息。

## 接口原型

- 请求方法：POST
- 请求地址：`https://aigc-chat-api.zegotech.cn?Action=DescribeCustomAgentTemplate`
- 传输协议：HTTPS
- 调用频率限制：10 次/秒
## 请求参数

| 参数 | 类型 | 是否必选 | 描述 |
|------|------|---------|------|
| AgentTemplateId | Array of String | 否 | 智能体模板 ID。<ul><li>不传：查询所有自定义智能体模版。</li><li>传：查询指定智能体模版。</li></ul> |
| PageNum | Int | 否 | 从第几页开始查询。默认为 1。 |
| PageSize | Int | 否 | 返回结果包含的模板详情数量上限。默认为 100。 |

## 请求示例

```json
{
    "AgentTemplateId": ["template_001", "template_002"],
    "PageNum": 1,
    "PageSize": 10
}
```

## 响应参数

| 参数 | 类型 | 描述 |
|------|------|------|
| Code | Number | 返回码。0 表示成功，其他值表示失败。如需了解错误码及响应处理建议，请参考 [返回码](./../Return%20codes.mdx)。 |
| Message | String | 返回信息描述。 |
| Data | Object | 返回数据。 |
|   └ Total | Int | 总数。 |
|   └ CustomAgentConfigList | Array of Object | 自定义智能体列表。本参数结构请见 [常用参数说明 - CustomAgentConfig](./../Common%20Parameter%20Description.mdx#customagentconfig)。 |
