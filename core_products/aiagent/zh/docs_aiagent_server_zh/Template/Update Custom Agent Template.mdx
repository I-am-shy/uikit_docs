# 修改模版
---

##  描述
本接口可用于修改一个自定义的智能体模版。

## 接口原型

- 请求方法：POST
- 请求地址：`https://aigc-chat-api.zegotech.cn?Action=UpdateCustomAgentTemplate`
- 传输协议：HTTPS
- 调用频率限制：10 次/秒
## 请求参数

| 参数 | 类型 | 是否必选 | 描述 |
|------|------|---------|------|
| CustomAgentConfig | Object | 是 | 自定义智能体参数。本参数结构，请见 [常用参数说明 - CustomAgentConfig](./../Common%20Parameter%20Description.mdx#customagentconfig)。 |

## 请求示例

- 请求地址 URL：

```https
https://aigc-chat-api.zegotech.cn?Action=UpdateCustomAgentTemplate
&<公共请求参数>
```

- 请求消息体：
```json
{
    "CustomAgentConfig": {
        "AgentTemplateId": "template_001",
        "Name": "升级版客服助手",
        "Avatar": "https://example.com/avatar_new.png",
        "Intro": "我是一个全新升级的客服助手",
        "System": "你是一个经过升级的专业客服助手，擅长解答各类复杂问题",
        "LLM": {
            "Type": "WenXin",
            "Model": "ERNIE-Bot-4",
            "IgnoreBracketText": [1],
            "ResponseFormatType": 0
        },
        "TTS": {
            "Type": "Aliyun",
            "Voice": "xiaoyun",
            "IgnoreBracketText": [1, 2]
        }
    }
}
```

## 响应参数

| 参数 | 类型 | 描述 |
|------|------|------|
| Code | Int | 返回码。0 表示成功，其他值表示失败。如需了解错误码及响应处理建议，请参考 [返回码](./../Return%20codes.mdx)。 |
| Message | String | 请求结果的说明信息。 |
| RequestId | String | 请求 ID。 |
## 响应示例

```json
{
    "Code": 0,
    "Message": "Succeed",
    "RequestId": "1843985617336143872",
    "Data": null
}
```
