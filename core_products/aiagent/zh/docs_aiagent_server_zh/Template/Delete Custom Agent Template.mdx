# 删除模版
---

##  描述

本接口可用于删除一个自定义的的智能体模板。

## 接口原型

- 请求方法：POST
- 请求地址：`https://aigc-chat-api.zegotech.cn?Action=DeleteCustomAgentTemplate`
- 传输协议：HTTPS
- 调用频率限制：10 次/秒
## 请求参数

| 参数 | 类型 | 是否必选 | 描述 |
|------|------|---------|------|
| AgentTemplateId | String | 是 | 由开发者自定义的智能体模板 ID。 |

## 请求示例

- 请求地址 URL：

```https
https://aigc-chat-api.zegotech.cn?Action=DeleteCustomAgentTemplate
&<公共请求参数>
```

- 请求消息体：
```json
{
    "AgentTemplateId": "template_001"
}
```

## 响应参数

| 参数 | 类型 | 描述 |
|------|------|------|
| Code | Int | 返回码。0 表示成功，其他值表示失败。如需了解错误码及响应处理建议，请参考 [返回码](./../Return%20codes.mdx)。 |
| Message | String | 请求结果的说明信息。 |
| RequestId | String | 请求 ID。 |
## 响应示例

```json
{
    "Code": 0,
    "Message": "success",
    "RequestId": "1898302332764950528"
}
```
