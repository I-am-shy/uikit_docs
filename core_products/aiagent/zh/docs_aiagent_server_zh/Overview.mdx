# API 概览

---

ZEGO 实时互动 AI Agent 服务端提供以下相关 API 接口及回调，可参考 [调用方式](./Accessing%20Server%20APIs.mdx) 调用以下接口。

## 会话管理

| 接口名称                                                            | 接口功能           | 默认调用频率限制 |
| ------------------------------------------------------------------- | ------------------ | ---------------- |
| [创建会话](./Conversation/Create%20Conversation.mdx)                | 为指定用户创建一个会话并配置智能体（Agent），用于后续进行 IM 与 RTC 对话。          | 10 次/秒         |
| [修改会话](./Conversation/Update%20Conversation.mdx)                | 更新会话配置。           | 10 次/秒         |
| [删除会话](./Conversation/Delete%20Conversation.mdx)                | 为指定用户删除一个会话。          | 10 次/秒         |
| [获取会话列表](./Conversation/Describe%20ConversationList.mdx)     | 获取指定用户的会话列表，或者查询单个会话的详细信息。       | 10 次/秒         |

## 会话消息管理

| 接口名称                                                            | 接口功能           | 默认调用频率限制 |
| ------------------------------------------------------------------- | ------------------ | ---------------- |
| [获取会话上下文消息](./Messages/Describe%20ConversationMsgList.mdx) | 获取指定会话中 LLM 使用的上下文。 | 10 次/秒         |
| [重置会话上下文消息](./Messages/Reset%20ConversationMsg.mdx)        | 重置指定会话中 LLM 使用的上下文。 | 10 次/秒         |

## RTC 语音对话管理

| 接口名称                                                            | 接口功能           | 默认调用频率限制 |
| ------------------------------------------------------------------- | ------------------ | ---------------- |
| [发起语音对话](./Voice%20Call/Start%20Rtc%20Chat.mdx)                     | 发起一次语音对话。    | 10 次/秒         |
| [结束语音对话](./Voice%20Call/Stop%20Rtc%20Chat.mdx)                       | 结束一次语音对话。   | 10 次/秒         |
| [主动调用 LLM](./Voice%20Call/Send%20Agent%20LLM%20Input.mdx)               | 以用户身份主动调用 LLM 服务，LLM 响应后发 TTS 给客户端。      | 10 次/秒         |
| [主动调用 TTS](./Voice%20Call/Send%20Agent%20TTS%20Input.mdx)               | 以智能体身份主动调用 TTS 服务，向用户发送语音消息。     | 10 次/秒         |

## 模板管理

| 接口名称                                                            | 接口功能           | 默认调用频率限制 |
| ------------------------------------------------------------------- | ------------------ | ---------------- |
| [创建模板](./Template/Create%20Custom%20Agent%20Template.mdx)                | 创建一个自定义的智能体模版。       | 10 次/秒         |
| [修改模板](./Template/Update%20Custom%20Agent%20Template.mdx)                | 修改一个自定义的智能体模版。  | 10 次/秒         |
| [删除模板](./Template/Delete%20Custom%20Agent%20Template.mdx)                | 删除一个自定义的的智能体模板。   | 10 次/秒         |
| [查询模板](./Template/Describe%20Custom%20Agent%20Template.mdx)     | 获取可用的智能体模板列表。       | 10 次/秒         |

## 回调

| 回调名称 | 回调说明 |
| -------- | -------- |
| [通用回调](./Callbacks/General%20Callback.mdx)         |   凭借此回调，监听用户与智能体对话过程中所发生的事件。       |
| [自定义 LLM System 回调](./Callbacks/Custom%20LLM%20System%20Callback.mdx)    |  凭借本回调，自定义本轮对话的 LLM System（即智能体的系统提示 Prompt）。        |
