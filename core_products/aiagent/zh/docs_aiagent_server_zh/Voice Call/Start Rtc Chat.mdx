# 发起语音对话
---

##  描述
本接口可用于发起一次语音对话。

## 接口原型

- 请求方法：POST
- 请求地址：`https://aigc-chat-api.zegotech.cn?Action=StartRtcChat`
- 传输协议：HTTPS
- 调用频率限制：10 次/秒
## 请求参数

<Note title="说明">

ID 类参数限制：
- 字符限制：数字、英文字符、'-'、'_'。
- 长度限制：
  - `UserId`：32 字节
  - `ConversationId`、`RoomId`、`StreamId`、`AgentStreamId`：128 字节。
- 请确保 `ConversationId`、`RoomId`、`StreamId` 以及 `AgentStreamId` 在同一 AppID 下全局唯一。
</Note>

| 参数 | 类型 | 是否必选 | 描述 |
|------|------|---------|------|
| UserId           | String | 是    | 用户 ID，                              |
| ConversationId   | String | 是    | 会话的唯一标识。           |
| RoomId | String | 是 | 房间 ID。 |
| StreamId | String | 是 | 用户音频流 ID（由开发者定义）。ZEGO 服务端根据此参数拉取用户语音。 |
| AgentStreamId | String | 是 | 智能体音频流 ID（由开发者定义）。ZEGO 服务端通过此参数推送机器人的语音。 |
| Window | Int | 否 | 对话上下文最大窗口。默认 20 条，最大 100 条。取值范围为 [0, 100]。 |

## 请求示例

- 请求地址 URL：

```https
https://aigc-chat-api.zegotech.cn?Action=StartRtcChat
&<公共请求参数>
```

- 请求消息体：
```json
{
    "UserId": "test_user_001",
    "ConversationId": "test_conversation_001",
    "RoomId": "test_room_001",
    "StreamId": "user_stream_001",
    "AgentStreamId": "agent_stream_001",
    "Window": 20
}
```

## 响应参数

| 参数 | 类型 | 描述 |
|------|------|------|
| Code | Int | 返回码。0 表示成功，其他值表示失败。如需了解错误码及响应处理建议，请参考 [返回码](./../Return%20codes.mdx)。 |
| Message | String | 返回信息描述。 |

## 响应示例

```json
{
    "Code": 0,
    "Message": "success"
}
```
