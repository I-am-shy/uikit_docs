# 结束语音对话
---

##  描述

本接口可用于结束一次语音对话。

## 接口原型

- 请求方法：POST
- 请求地址：`https://aigc-chat-api.zegotech.cn?Action=StopRtcChat`
- 传输协议：HTTPS
- 调用频率限制：10 次/秒
## 请求参数

| 参数 | 类型 | 是否必选 | 描述 |
|------|------|---------|------|
| UserId           | String | 是    | 用户 ID。仅支持数字、英文字符、'-'、'_'。长度上限为 32 字节。                              |
| ConversationId   | String | 是    | 会话的唯一标识。仅支持数字、英文字符、'-'、'_'。长度上限为 128 字节。在同一 AppID 下全局唯一。           |

## 请求示例

- 请求地址 URL：

```https
https://aigc-chat-api.zegotech.cn?Action=StopRtcChat
&<公共请求参数>
```

- 请求消息体：
```json
{
    "UserId": "test_user_001",
    "ConversationId": "test_conversation_001"
}
```

## 响应参数

| 参数 | 类型 | 描述 |
|------|------|------|
| Code | Int | 返回码。0 表示成功，其他值表示失败。如需了解错误码及响应处理建议，请参考 [返回码](./../Return%20codes.mdx)。 |
| Message | String | 请求结果的说明信息。 |
| RequestId | String | 请求 ID。 |
## 响应示例

```json
{
    "Code": 0,
    "Message": "success",
    "RequestId": "1898302332764950528"
}
```
