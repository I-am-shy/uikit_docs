# 关键概念

---

### AI Agent（智能体）

AI Agent（智能体）代表创建的 AI 角色。这个角色的参数包含了名称、头像、简介、提示词、音色等信息，以及使用的 LLM、TTS、ASR 信息等。

相关字段：`CustomAgentConfig`。

### 模版库

模版库是所有 AI Agent 模板的集合。开发者可以通过 [服务端 API](/aiagent-server/template/describe-custom-agent-template) 接口查询所有可用的模板。

### 会话（conversation）

用户与 AI Agent 创建的会话。一个用户可以选定一个 AI Agent 进行会话。会话的历史记录中，最近的若干条历史消息会被作为上下文供 LLM 使用。

用户可以多次进入或退出会话。

相关字段示例：`ConversationId`。

### 对话

在一个与 AI Agent 的会话中，支持 IM 对话、语音对话和数字人对话，但是同一时间只能进行一种对话。

### 历史记录

用户与 AI Agent 的所有历史消息。

### 上下文

当前会话中大模型使用的最近若干条历史消息，是历史记录的子集。支持开发者重置上下文，用以开启新话题。
