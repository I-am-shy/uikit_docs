# 发布日志

---

## 2.0.0 版本
发布日期：2025-04-24

- 接入体验升级，通过不到 10 行代码，即可实现与 AI 智能体的语音通话。
- 全流程音频处理能力升级，大幅优化语音打断及识别准确率，尤其是在噪音环境、播放 BGM、双讲（AI 和用户同时讲话）等场景，全面覆盖居家、办公室、公共环境等各种环境下的 AI 互动。
- 同步支持包括：自定义三方大语言模型（LLM）、500ms 内的自然语音打断、实时字幕、智能体状态查询、主动调用 LLM、主动调用 TTS 等能力。
- 功能架构升级：支持多用户 vs 多AI智能体架构，更灵活的互动形式。

---

## 1.4.0 版本

发布日期：2025-03-21

<h5>新增功能</h5>

- 新增查询智能体状态接口。
- 创建会话时，TTS 配置对象新增透传第三方请求参数 `ExtensionTTSParams`。
- 适用于 Minimax 的 TTS 透传参数 `ExtensionTTSParams` 新增 `model` 字段。
- ASR 配置对象新增热词 `HotWordId` 和扩展参数 `ExtensionASRParams`。
- 用于主动调用 TTS 服务的接口的请求参数新增字段 `RemoveMsgHistory`。

## 1.2.0 版本

发布日期：2025-01-16

<h5>新增功能</h5>

- 创建会话在 LLM 新增 `ResponseFormatType`、`ResponseMessageName` 字段。
- 会话、对话相关接口和用于主动调用 LLM 和 TTS 的接口的请求参数新增必选字段 `UserId`。
- TTS 配置对象中的扩展参数 `ExtensionParams` 新增字段 `ApiType` 和 `ResourceId`。

---

## 1.1.0 版本

发布日期：2025-01-08

<h5>新增功能</h5>

- `DescribeConversationList` 接口新增 `ConversationId` 字段，支持根据会话 ID 查询会话详情。
- `CreateConversation` 接口新增 `ChatHistoryMode` 字段，支持是否保存会话历史消息。

<h5>改进优化</h5>
- 调整房间事件消息协议。

<h5>废弃删除</h5>
- 删除 LLM、TTS 配置对象中的 `AccountSource` 字段。

## 1.0.0 版本

发布日期：2024-12-31

1.0.0 版本正式发布，优势如下所示：
- 全面的服务可靠性 & 稳定性。
- 更低的端到端延迟、打断延迟。
- 更新音频处理能力，支持嘈杂环境，满足 80% 以上场景覆盖。
- 智能体模版库。
- 支持主动调用 LLM。
- 支持主动调用 TTS。
- 支持自定义 RAG 等能力。
- LLM 配置对象新增 `IgnoreBracketText` 字段，支持过滤 LLM 文本的表情包

## Beta 0.5.0 版本

发布日期：2024-12-16

<h5>新增功能</h5>

- 新增用于主动调用 TTS 服务的接口。
- 新增用于主动调用 LLM 服务的接口。

## Beta 0.3.0 版本
