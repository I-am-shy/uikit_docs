#  重置会话上下文消息
---

##  描述
重置当前会话中大模型使用的上下文消息。

## 接口原型

- 请求方法：POST
- 请求地址：`https://aigc-chat-api.zegotech.cn?Action=ResetConversationMsg`
- 传输协议：HTTPS

## 请求参数

| 参数 | 类型 | 是否必选 | 描述 |
|------|------|---------|------|
| UserId | String | 是 | 用户id。 |
| ConversationId | String | 是 | 会话的唯一标识 |
