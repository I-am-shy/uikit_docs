# 自定义调用TTS
---

##  描述

自定义调用TTS发送消息给客户端。
- 当调用此接口时，若Agent正在说话，则会进行打断；
- 调用的信息会记录在会话消息历史记录中，但不作为上下文输入给 LLM。

## 接口原型

- 请求方法：POST
- 请求地址：`https://aigc-chat-api.zegotech.cn?Action=SendAgentTTSInput`
- 传输协议：HTTPS
- 调用频率限制：10 次/秒
## 请求参数

| 参数 | 类型 | 是否必选 | 描述 |
|------|------|---------|------|
| UserId | String | 是 | 用户id。 |
| ConversationId | String | 是 | 会话的唯一标识。 |
| AgentId | String | 是 | AgentId。机器人id，会被用于登录IM\RTC。 |
| Text | String | 是 | 发送给 TTS 的文本内容，最大不超过 300 个字符。 |

## 请求示例

```json
{
  "UserId": "test_user_001",
  "ConversationId": "test_conversation_001",
  "AgentId": "@RBT#test_agent_001",
  "Text": "欢迎使用我们的服务,有什么可以帮您?"
}
```

## 响应参数

| 参数 | 类型 | 描述 |
|------|------|------|
| Code | Int | 返回码。0 表示成功，其他值表示失败。如需了解错误码及响应处理建议，请参考 [返回码](./../Return%20codes.mdx)。 |
| Message | String | 返回信息描述。 |

## 响应示例

```json
{
    "Code": 0,
    "Message": "success"
}
```
