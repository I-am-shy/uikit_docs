# 发起语音对话
---

##  描述
当需要进行语音对话时，用于发起一次语音对话。

## 接口原型

- 请求方法：POST
- 请求地址：`https://aigc-chat-api.zegotech.cn?Action=StartRtcChat`
- 传输协议：HTTPS
- 调用频率限制：10 次/秒
## 请求参数

| 参数 | 类型 | 是否必选 | 描述 |
|------|------|---------|------|
| UserId | String | 是 | 用户id。 |
| ConversationId | String | 是 | 会话的唯一标识。需要保证全局唯一。 |
| RoomId | String | 是 | 房间id。主要保证App下RoomId唯一 |
| StreamId | String | 是 | 流id。用户音频流，开发者来指定，服务端根据这个StreamId来拉取用户语音。主要保证App下用户StreamId唯一 |
| AgentStreamId | String | 是 | 机器人流id，开发者来指定，服务端把机器人的语音推送到这个AgentStreamId。主要保证App下机器人StreamId唯一 |
| Window | Int | 否 | 对话上下文最大窗口。默认 20 条，最大 100 条。 |

## 请求示例

```json
{
  "UserId": "test_user_001",
  "ConversationId": "test_conversation_001",
  "RoomId": "test_room_001",
  "StreamId": "user_stream_001",
  "AgentStreamId": "agent_stream_001",
  "Window": 20
}
```

## 响应参数

| 参数 | 类型 | 描述 |
|------|------|------|
| Code | Int | 返回码。0 表示成功，其他值表示失败。 |
| Message | String | 返回信息描述。 |

## 响应示例

```json
{
    "Code": 0,
    "Message": "success"
}
```
