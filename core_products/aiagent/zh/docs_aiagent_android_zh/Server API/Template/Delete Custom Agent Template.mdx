# 删除模版
---

##  描述
删除已存在的自定义Agent模版。

## 接口原型

- 请求方法：POST
- 请求地址：`https://aigc-chat-api.zegotech.cn?Action=DeleteCustomAgentTemplate`
- 传输协议：HTTPS

## 请求参数

| 参数 | 类型 | 是否必选 | 描述 |
|------|------|---------|------|
| AgentTemplateId | String | 是 | Agent模版id。 |

## 请求示例

```json
{
  "AgentTemplateId": "template_001"
}
```

## 响应参数

| 参数 | 类型 | 描述 |
|------|------|------|
| Code | Int | 返回码。0表示成功，其他值表示失败 |
| Message | String | 返回信息描述 |

## 响应示例

```json
{
    "Code": 0,
    "Message": "success"
}
```
