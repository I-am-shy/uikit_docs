# 修改模版
---

##  描述
更新已存在的自定义Agent模版。

## 接口原型

- 请求方法：POST
- 请求地址：`https://aigc-chat-api.zegotech.cn?Action=UpdateCustomAgentTemplate`
- 传输协议：HTTPS
- 调用频率限制：10 次/秒
## 请求参数

| 参数 | 类型 | 是否必选 | 描述 |
|------|------|---------|------|
| CustomAgentConfig | Object | 是 | 自定义Agent参数。<br />用于自定义Agent的各种参数。<br />见CustomAgentConfig |

## 请求示例

```json
{
  "CustomAgentConfig": {
    "AgentTemplateId": "template_001",
    "Name": "升级版客服助手",
    "Avatar": "https://example.com/avatar_new.png",
    "Intro": "我是一个全新升级的客服助手",
    "System": "你是一个经过升级的专业客服助手，擅长解答各类复杂问题",
    "LLM": {
      "Type": "WenXin",
      "Model": "ERNIE-Bot-4",
      "IgnoreBracketText": [1],
      "ResponseFormatType": 0
    },
    "TTS": {
      "Type": "Aliyun",
      "Voice": "xiaoyun",
      "IgnoreBracketText": [1, 2]
    }
  }
}
```

## 响应参数

| 参数 | 类型 | 描述 |
|------|------|------|
| Code | Int | 返回码。0 表示成功，其他值表示失败。 |
| Message | String | 返回信息描述。 |

## 响应示例

```json
{
    "Code": 0,
    "Message": "success"
}
```
