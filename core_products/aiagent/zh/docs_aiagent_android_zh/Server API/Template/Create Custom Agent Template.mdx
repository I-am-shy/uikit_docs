# 创建模版
---

##  描述
服务端可以创建自定义的Agent模版配置，在创建会话时指定模版ID即可使用。

## 接口原型

- 请求方法：POST
- 请求地址：`https://aigc-chat-api.zegotech.cn?Action=CreateCustomAgentTemplate`
- 传输协议：HTTPS

## 请求参数


| 参数 | 类型 | 是否必选 | 描述 |
|------|------|---------|------|
| CustomAgentConfig | Object | 是 | 自定义Agent参数。<br />用于自定义Agent的各种参数。<br />见CustomAgentConfig |
