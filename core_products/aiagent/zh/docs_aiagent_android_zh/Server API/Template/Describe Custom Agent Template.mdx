# 查询模版
---

##  描述
查询可用的Agent模版。

## 接口原型

- 请求方法：POST
- 请求地址：`https://aigc-chat-api.zegotech.cn?Action=DescribeCustomAgentTemplate`
- 传输协议：HTTPS

## 请求参数

| 参数 | 类型 | 是否必选 | 描述 |
|------|------|---------|------|
| AgentTemplateId | []String | 否 | Agent模版id。<br />App下唯一。<br />不传则查询App下所有自定义Agent模版 |
| PageNum | Int | 否 | 页码。默认为1。 |
| PageSize | Int | 否 | 页宽。默认为100。 |

## 请求示例

```json
{
  "AgentTemplateId": ["template_001", "template_002"],
  "PageNum": 1,
  "PageSize": 10
}
```

## 响应参数

| 参数 | 类型 | 描述 |
|------|------|------|
| Total | Int | 总数。 |
| CustomAgentConfigList | []Object | 自定义Agent列表。<br />见CustomAgentConfig |
