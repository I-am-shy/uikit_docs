# 修改会话
---

##  描述
更新会话配置，如果存在进行中的IM与RTC对话，Agent配置实时生效。

## 接口原型

- 请求方法：POST
- 请求地址：`https://aigc-chat-api.zegotech.cn?Action=UpdateConversation`
- 传输协议：HTTPS

## 请求参数

| 参数               | 类型   | 是否必选 | 描述                                                                                                                                   |
|------------------|------|------|--------------------------------------------------------------------------------------------------------------------------------------|
| UserId           | String | 是    | 用户id。                                                                                                                             |
| ConversationId   | String | 是    | 会话的唯一标识。需要保证全局唯一。                                                                                                               |
| AgentTemplateId  | String | 否    | Agent模版id。zego提供或用户自定义，对应一个默认的agent配置。（暂不支持用户自定义）AgentTemplateId与CustomAgentConfig必须传一个。                                           |
| CustomAgentConfig| Object | 否    | 自定义Agent参数。用于自定义Agent的各种参数。若AgentTemplateId不为空，则在该模版基础上，根据CustomAgentConfig中传入的参数做修改。见CustomAgentConfig                  |

## 请求示例

```json
{
  "UserId": "test_user_001",
  "ConversationId": "test_conversation_001",
  "AgentTemplateId": "template_001",
  "CustomAgentConfig": {
    "Name": "AI助手",
    "Avatar": "https://example.com/avatar.png",
    "Intro": "我是一个升级后的AI助手",
    "System": "你是一个更专业的AI助手",
    "LLM": {
      "Type": "WenXin",
      "Model": "ERNIE-Bot-4",
      "IgnoreBracketText": [1],
      "ResponseFormatType": 0
    }
  }
}
```

## 响应参数

| 参数 | 类型 | 描述 |
|------|------|------|
| Code | Int | 返回码。0表示成功，其他值表示失败 |
| Message | String | 返回信息描述 |

## 响应示例

```json
{
    "Code": 0,
    "Message": "success"
}
```
