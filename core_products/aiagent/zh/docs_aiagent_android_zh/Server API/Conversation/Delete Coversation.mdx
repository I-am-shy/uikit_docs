# 删除会话
---

##  描述
删除会话配置。

## 接口原型

- 请求方法：POST
- 请求地址：`https://aigc-chat-api.zegotech.cn?Action=DeleteConversation`
- 传输协议：HTTPS
- 调用频率限制：10 次/秒

## 请求参数

| 参数 | 类型 | 是否必选 | 描述 |
|------|------|---------|------|
| UserId | String | 是 | 用户id。 |
| ConversationId | String | 是 | 会话的唯一标识 |

## 请求示例

```json
{
  "UserId": "test_user_001",
  "ConversationId": "test_conversation_001"
}
```

## 响应参数

| 参数 | 类型 | 描述 |
|------|------|------|
| Code | Int | 返回码。0表示成功，其他值表示失败 |
| Message | String | 返回信息描述 |

## 响应示例

```json
{
    "Code": 0,
    "Message": "success"
}
```
