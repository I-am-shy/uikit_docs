# 获取会话列表
---

##  描述
获取会话列表，APP可以用做维护已存在的会话列表或用ZIM SDK。

## 接口原型

- 请求方法：POST
- 请求地址：`https://aigc-chat-api.zegotech.cn?Action=DescribeConversationList`
- 传输协议：HTTPS

## 请求参数

| 参数 | 类型 | 是否必选 | 描述 |
|------|------|---------|------|
| UserId | String | 是 | 用户id。 |
| ConversationId | String | 否 | 会话的唯一标识 |
| PageNum | Int | 否 | 页码。默认为1。 |
| PageSize | Int | 否 | 页宽。默认为100。 |

## 请求示例

```json
{
  "UserId": "test_user_001",
  "ConversationId": "test_conversation_001",
  "PageNum": 1,
  "PageSize": 10
}
```

## 响应参数

| 参数 | 类型 | 描述 |
|------|------|------|
| Total | Int | 会话总数。 |
| ConversationList | []Object | 会话详情。见ConversationInfo |


**ConversationInfo:**

| 参数 | 类型 | 描述 |
|------|------|------|
| ConversationId | String | 会话的唯一标识。 |
| UserId | String | 用户id。 |
| AgentId | String | AgentId。 |
| AgentTemplateId | String | Agent模版id。 |
| CustomAgentConfig | Object | 自定义Agent参数。同CustomAgentConfig |
| IsChatting | Bool | 是否存在进行中的对话。 |
