# 自定义调用LLM
---

##  描述

模拟用户输入信息，自定义发送消息给LLM，LLM响应后发TTS给客户端。
- 当调用此接口时，若Agent正在说话，则会进行打断；
- 调用的信息会记录在会话消息历史记录中，会作为上下文输入给 LLM。

## 接口原型

- 请求方法：POST
- 请求地址：`https://aigc-chat-api.zegotech.cn?Action=SendAgentLLMInput`
- 传输协议：HTTPS

## 请求参数

| 参数 | 类型 | 是否必选 | 描述 |
|------|------|---------|------|
| UserId | String | 是 | 用户id。 |
| ConversationId | String | 是 | 会话的唯一标识。 |
| AgentId | String | 是 | AgentId。机器人id，会被用于登录IM\RTC。 |
| Text | String | 是 | 发送给 LLM 的文本内容，最大不超过 300 个字符。 |
| System | String | 否 | 是否需要临时修改本次对话的System。若填空，则不改动本轮对话的system |
