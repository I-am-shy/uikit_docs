# 自定义调用TTS
---

##  描述

自定义调用TTS发送消息给客户端。
- 当调用此接口时，若Agent正在说话，则会进行打断；
- 调用的信息会记录在会话消息历史记录中，但不作为上下文输入给 LLM。

- 请求地址：
- 传输协议：HTTPS

## 请求参数

| 参数 | 类型 | 是否必选 | 描述 |
|------|------|---------|------|
| UserId | String | 是 | 用户id。 |
| ConversationId | String | 是 | 会话的唯一标识。 |
| AgentId | String | 是 | AgentId。机器人id，会被用于登录IM\RTC。 |
| Text | String | 是 | 发送给 TTS 的文本内容，最大不超过 300 个字符。 |
