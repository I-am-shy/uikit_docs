# Callback

## AIAgent 回调接口

### 会话管理回调接口

#### CreateConversationCallback

创建会话回调接口。

参数说明：
- errorCode: 错误码
  - 0: 表示成功
  - 非0: 表示失败
- message: 结果描述信息
  - 成功时可能为空
  - 失败时包含具体错误信息
- requestId: 请求ID，用于标识本次请求
- conversation: 创建的会话对象

<CodeGroup>
```java title="函数原型"
void onResult(int errorCode, String message, String requestId, Conversation conversation);
```
</CodeGroup>
