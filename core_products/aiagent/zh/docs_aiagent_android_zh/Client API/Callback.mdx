# Callback

## AIAgent 回调接口

### 会话管理回调接口

#### CreateConversationCallback

创建会话回调接口。

```java
void onResult(int errorCode, String message, String requestId, Conversation conversation);
```

#### DeleteConversationCallback

删除会话回调接口。

```java
void onResult(int errorCode, String message, String requestId);
```

#### QueryConversationListCallback

查询会话列表回调接口。

```java
void onResult(int errorCode, String message, String requestId, QueryConversationListData conversationListData);
```

#### UpdateConversationCallback

更新会话配置回调接口。

```java
void onResult(int errorCode, String message, String requestId);
```

#### ResetConversationMsgCallback

重置会话上下文回调接口。

```java
void onResult(int errorCode, String message, String requestId);
```

### 语音/视频通话回调接口

#### StartRTCChatCallback

开始语音通话回调接口。

```java
void onResult(int errorCode, String message, String requestId);
```

#### StartRTCVideoChatCallback

开始视频通话回调接口。

```java
void onResult(int errorCode, String message, String requestId);
```

#### StopRTCChatCallback

停止通话回调接口。

```java
void onResult(int errorCode, String message, String requestId);
```
