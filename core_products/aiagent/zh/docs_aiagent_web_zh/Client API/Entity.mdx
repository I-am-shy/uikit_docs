# Entity

## AIAgentResult

AIAgentResult 接口用于表示AI智能体操作的结果，包含以下属性：
- `code`: 结果码
- `message`: 结果信息
- `requestID`: 请求ID（可选）

示例用法：
```typescript
const result: AIAgentResult = {
  code: 0,
  message: "操作成功",
  requestID: "req_123456"
};
```

## AIAgentCreateResult

AIAgentCreateResult 接口继承自 AIAgentResult，用于表示创建会话的结果，额外包含以下属性：
- `conversation`: 创建的会话对象（可选）

示例用法：
```typescript
const createResult: AIAgentCreateResult = {
  code: 0,
  message: "会话创建成功",
  requestID: "req_123456",
  conversation: {
    conversationID: "conv_123",
    userID: "user_001",
    agentID: "agent_001",
    agentTemplateID: "template_001",
    customAgentConfig: {...},
    chatConfig: {...},
    isChatting: false
  }
};
```


## Conversation

Conversation 接口用于描述会话信息，包含以下属性：
- `conversationID`: 会话ID
- `userID`: 用户ID
- `agentID`: AI智能体ID
- `agentTemplateID`: AI智能体模板ID
- `customAgentConfig`: AI智能体自定义配置
- `chatConfig`: 聊天配置
- `isChatting`: 是否正在聊天（可选）

示例用法：
```typescript
const conversation: Conversation = {
  conversationID: "conv_123",
  userID: "user_001",
  agentID: "agent_001",
  agentTemplateID: "template_001",
  customAgentConfig: {
    AgentTemplateId: "template_001",
    Name: "AI助手",
    Avatar: "https://example.com/avatar.png",
    Intro: "我是一个AI助手",
    System: "你是一个专业的AI助手"
  },
  chatConfig: {
    LLMSystemMode: 0,
    EnableLLMServerMessage: 1
  },
  isChatting: false
};
```

## CustomAgentConfig

CustomAgentConfig 接口用于配置AI智能体，包含以下属性：
- `AgentTemplateId`: AI智能体模板ID
- `Name`: AI智能体名称
- `Avatar`: AI智能体头像URL
- `Intro`: AI智能体简介
- `System`: AI智能体系统提示词
- `LLM`: 大语言模型配置（可选）
- `TTS`: 语音合成配置（可选）
- `ASR`: 语音识别配置（可选）
- `WelcomeMessage`: AI智能体欢迎语（可选）
- `Sex`: AI智能体性别（可选）
- `MetaHumanConfig`: 数字人配置（可选）

示例用法：
```typescript
const customAgentConfig: CustomAgentConfig = {
  AgentTemplateId: "template_001",
  Name: "AI助手",
  Avatar: "https://example.com/avatar.png",
  Intro: "我是一个专业的AI助手",
  System: "你是一个专业的AI助手",
  LLM: {
    Type: "gpt-3.5",
    Model: "gpt-3.5-turbo",
    ResponseFormatType: 0,
    ResponseMessageName: "AI助手"
  },
  TTS: {
    Type: "standard",
    Voice: "female_01"
  },
  ASR: {
    HotWordId: "hotword_001"
  },
  WelcomeMessage: "你好，我是AI助手",
  Sex: "female",
  MetaHumanConfig: {
    MetaHuman: {
      MetaHumanId: "mh_001",
      Layout: {
        Top: 0,
        Left: 0,
        Width: 1280,
        Height: 720,
        Layer: 1
      }
    }
  }
};
```

```typescript
      Top: 0,
      Left: 0,
      Width: 1280,
      Height: 720,
      Layer: 1
    }
  },
  BackgroundColor: "#FFFFFF",
  Driver: {
    DriverType: 0,
    PitchRate: 1.0,
    SpeechRate: 1.0,
    Volume: 1.0,
    TimbreId: "timbre_001"
  },
  VideoOption: {
    Width: 1280,
    Height: 720,
    Format: "mp4"
  },

    Enable: 1
  }
};
```

## UpdateConversationConfig

UpdateConversationConfig 接口用于更新会话配置，包含以下属性：
- `agentTemplateID`: AI智能体模板ID（可选）
- `customAgentConfig`: AI智能体自定义配置（可选）
- `chatConfig`: 聊天配置（可选）

示例用法：
```typescript
const updateConfig: UpdateConversationConfig = {
  agentTemplateID: "new_template_001",
  customAgentConfig: {
    Name: "新AI助手",
    Avatar: "https://example.com/new_avatar.png"
  },
  chatConfig: {
    LLMSystemMode: 1,
    EnableLLMServerMessage: 1
  }
};
```

## LLMSystemMode

LLMSystemMode 枚举用于定义LLM系统模式：
- `Default`: 使用会话配置CustomAgentConfig的System
- `ASRParallel`: 优先使用会话配置CustomAgentConfig的System；并行业务基于ASR识别结果自定义本轮对话System
- `ASRWithTimeout`: 业务基于ASR识别结果自定义本轮对话System，5s超时后使用默认System

示例用法：
```typescript
const mode: LLMSystemMode = LLMSystemMode.Default; // 0
```

## EnableLLMServerMessage

EnableLLMServerMessage 枚举用于定义是否启用LLM服务器消息：
- `Disable`: 不启用
- `Enable`: 启用

示例用法：
```typescript
const enable: EnableLLMServerMessage = EnableLLMServerMessage.Enable; // 1
```

## ChatMode

ChatMode 枚举用于定义聊天模式：
- `Voice`: 语音模式
- `MetaHuman`: 数字人模式

示例用法：
```typescript
const mode: ChatMode = ChatMode.Voice; // 0
```
