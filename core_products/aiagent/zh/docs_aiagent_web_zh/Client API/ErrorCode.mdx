# 错误码

## AIAgentError

AIAgentError 枚举定义了 SDK 可能遇到的各种错误情况及其对应的错误码。

### 基础错误码

| 错误码 | 说明 |
|--------|------|
| `Success = 0` | 操作成功 |
| `InvalidParams = 900000001` | 参数无效 |
| `NetworkError = 900000002` | 网络错误 |
| `InvalidToken = 900000003` | 令牌无效 |
| `ServiceUnknown = 900000004` | 服务未知错误 |
| `NetworkTimeout = 900000005` | 网络超时 |
| `ConversationNotExist = 900000006` | 会话不存在 |
| `ConversationNotSupportMetaHuman = 900000007` | 该会话不支持数字人 |

### 网关错误码

| 错误码 | 说明 |
|--------|------|
| `InvalidHost = 100000000` | Host错误 |
| `InvalidAppId = 100000001` | AppId格式错误 |
| `EmptyTimestamp = 100000002` | Timestamp为空 |
| `InvalidTimestamp = 100000003` | Timestamp格式错误 |
| `SignatureExpired = 100000004` | 签名过期 |
| `InvalidSignature = 100000005` | 签名错误 |
| `EmptyAction = 100000006` | Action为空 |
| `UnsupportedAction = 100000007` | 不支持的Action |
| `EmptySignatureNonce = 100000008` | SignatureNonce为空 |
| `EmptySignature = 100000009` | Signature为空 |
| `GetServerSecretFailed = 100000010` | 获取ServerSecret失败 |
| `GetAppConfigFailed = 100000011` | 获取App配置失败 |
| `ReadHttpBodyFailed = 100000012` | 读取HTTP包体失败 |
| `EmptySignatureVersion = 100000013` | SignatureVersion为空 |
| `UnsupportedSignatureVersion = 100000014` | 不支持的SignatureVersion |
| `GatewayInternalError = 100000015` | 网关内部错误 |
| `AppEnvNotConfigured = 100000016` | App未配置正式环境 |
| `ServiceNotActivated = 100000017` | App未开通服务 |

## 示例用法

```typescript
// 检查操作结果
if (result.code === AIAgentError.Success) {
  console.log('操作成功');
} else if (result.code === AIAgentError.InvalidParams) {
  console.error('参数无效');
} else if (result.code === AIAgentError.NetworkTimeout) {
  console.error('网络超时，请重试');
}

// 处理业务错误
switch (error.code) {
  case AIAgentError.ServiceNotActivated:
    console.error('服务未开通，请先开通服务');
    break;
  case AIAgentError.AppConcurrentLimitReached:
    console.error('已达到并发限制，请稍后重试');
    break;
  case AIAgentError.UnsupportedTtsProvider:
    console.error('当前TTS供应商不支持');
    break;
  default:
    console.error('未知错误：', error.message);
}
```
