## 核心能力实现




### 初始化用户信息和 SDK 参数

<Warning title="注意">
在实现 AI Agent 的功能之前，请确保已经集成了 [ZEGO Express Web SDK](https://doc-zh.zego.im/article/199)。
</Warning>

1. 引入SDK并获取实例： 【省略RTC】

【以下流程添加说明使用的API（包括RTC）和实现的功能，例如，使用getInstance获取ZegoAIAgent的单例实例。】
```typescript
import { ZegoAIAgent } from 'zego-AIAgent';

const AIAgent = ZegoAIAgent.getInstance();

// 配置认证信息
const authData = {
    appID: your_app_id,    // 您的应用ID
    token: 'your_token'    // 您的认证Token
};

// 配置用户信息
const userProfile = {
    userID: 'user123',     // 用户ID
    userName: 'username'   // 用户名称
};

// 初始化SDK
try {
    const result = await AIAgent.initSDK(authData, userProfile);
    if (result.code === 0) {
        console.log('SDK初始化成功');
    }
} catch (error) {
    console.error('SDK初始化失败:', error);
}
```

### 创建会话

```typescript
const customAgentConfig = {
    agentTemplateId: 'template_id',  // AI智能体模板ID
    name: 'AI助手',                  // AI智能体名称
    intro: 'AI助手简介',            // 简介
    system: '系统提示词',           // 系统提示词
    welcomeMessage: '您好，我是AI助手', // 欢迎语
    sex: 'female',                  // AI性别
    // 大语言模型配置
    llm: {
        type: 'llm_type',
        model: 'model_name',
        responseFormatType: 0,
        responseMessageName: 'assistant'
    },
    // 语音合成配置
    tts: {
        type: 'tts_type',
        voice: 'voice_type'
    },
    // 语音识别配置
    asr: {
        HotWordId: 'hot_word_id'
    }
};

// 创建会话
try {
    const createResult = await AIAgent.createConversation(customAgentConfig);
    if (createResult.code === 0 && createResult.conversation) {
        const conversationID = createResult.conversation.conversationID;
        console.log('会话创建成功，会话ID:', conversationID);
    }
} catch (error) {
    console.error('会话创建失败:', error);
}
```
### 实现 AI 语音互动
```typescript
// 开始语音聊天
const voiceChatOptions = {
    conversationID: 'conversation_id',  // 会话ID
    roomID: 'room_id',                 // 房间ID
    userStreamID: 'user_stream_id',    // 用户流ID
    agentStreamID: 'agent_stream_id',  // 智能体流ID
    chatMode: ChatMode.Voice           // 聊天模式：语音
};

try {
    const startResult = await AIAgent.startChat(voiceChatOptions);
    if (startResult.code === 0) {
        console.log('语音聊天开始成功');
    }
} catch (error) {
    console.error('开始语音聊天失败:', error);
}
```
### 结束会话
```typescript
try {
    const stopResult = await AIAgent.stopChat('conversation_id');
    if (stopResult.code === 0) {
        console.log('聊天结束成功');
    }
} catch (error) {
    console.error('结束聊天失败:', error);
}
```
