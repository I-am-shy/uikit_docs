
## 响应参数

| 参数 | 类型 | 描述 |
|------|------|------|
| Code | Number | 返回码。0 表示成功，其他值表示失败。如需了解错误码及响应处理建议，请参考 [返回码](./../Return%20codes.mdx)。 |
| Message | String | 请求结果的说明信息。 |
| RequestId | String | 请求 ID。 |
| Data | Object | 返回数据。 |
| └TaskList | Array of Object | 正在运行中的数字人视频流任务列表。 |
| &nbsp;&nbsp;└TaskId      | String       | 数字人视频流任务 ID。 |
| &nbsp;&nbsp;└RoomId      | String       | 数字人视频流任务 的房间ID。[创建数字人实时视频流任务](./Create%Digital%20Human%20Stream%20Task.mdx) 传入的房间 ID。 |
| &nbsp;&nbsp;└TaskId      | String       | 数字人视频流任务 的流ID。[创建数字人实时视频流任务](./Create%Digital%20Human%20Stream%20Task.mdx) 传入的流 ID。 |
| &nbsp;&nbsp;└Status      | Number       | 任务状态：<br/>1：视频流任务初始化中。<br/>3：推流中。 |
| &nbsp;&nbsp;└CreateTime  | Number       | 任务创建的 Unix 时间戳 （秒）。 |
