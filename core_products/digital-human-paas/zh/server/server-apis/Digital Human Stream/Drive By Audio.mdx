# 音频驱动数字人

---

##  描述

通过本接口，您可以使数字人播报指定的输入音频。

## 请求参数

以下请求参数列表仅列出了接口请求参数和部分公共参数，完整公共参数列表请参考 [调用方式 - 公共请求参数](./../Accessing%20Server%20APIs.mdx#公共请求参数)。

| 参数               | 类型   | 是否必选 | 描述             |
|------------------|------|------|-------------------------|
| TaskId      | String | 是    | 数字人实时视频流任务 ID。 通过 [创建数字人实时视频流任务](./Create%20Digital%20Human%20Stream%20Task.mdx) 接口的响应参数获取。    |
| AudioUrl      | String | 是    | 音频 URL 链接（音频格式支持 MP3 和 WAV 文件）。    |
| InterruptMode      | Number | 否    | 打断模式：<br/>0：排队等待前面的播报任务完成后，再执行本播报任务。<br/>1：中断当前播报任务，立即执行本播报任务。 <br/>如果不填，取值为 0。   |

## 请求示例

- 请求地址 URL：

    ```https
    https://aigc-digitalhuman-api.zegotech.cn?Action=DriveByAudio
    &<公共请求参数>
    ```

- 请求消息体：

    ```json
    {
        "TaskId": "a46daa97-276a-409b-9bde-4146e2bb0041",
        "AudioUrl": "https://zego-aigc-test.oss-accelerate.aliyuncs.com/audio/20250612/5b3819fe-c507-45f7-a5c3-a4f817e71c8b.wav?OSSAccessKeyId=LTAI5tGqv9rje9rHtTDWyDXo&Expires=1749801042&Signature=aiI80PCICHuEFeUl4W5wITw86V8%3D",
        "InterruptMode": 1
    }
    ```

## 响应参数

| 参数 | 类型 | 描述 |
|------|------|------|
| Code | Number | 返回码。0 表示成功，其他值表示失败。如需了解错误码及响应处理建议，请参考 [返回码](./../Return%20codes.mdx)。 |
| Message | String | 请求结果的说明信息。 |
| RequestId | String | 请求 ID。 |
| └DriveId      | String       | 数字人实时视频流驱动任务 ID。 |
