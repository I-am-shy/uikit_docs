

# 查询数字人模特列表

---

## 描述

调用本接口查询指定 AppID 下可用的数字人模特列表。


## 接口原型

- 请求方法：POST
- 中国大陆正式环境请求地址：https://aigc-api.zegotech.cn/?Action=DescribeMetaHumanModel
- 传输协议：HTTPS
- 调用频率限制：10 次/秒


## 请求参数

本接口只需传入公共参数，公共参数列表请参考 [调用方式 - 公共请求参数\|_blank](#18534#4_1)。

## 请求示例

```
https://aigc-api.zegotech.cn/?Action=DescribeMetaHumanModel
&AppId=1234567890
&SignatureNonce=15215528852396
&Timestamp=1234567890
&Signature=7a2c0f11145fb760d607a07b54825013
&SignatureVersion=2.0
```
