

# 查询音色列表

---

## 描述

调用本接口查询数字人形象可用的音色列表。

<Note title="Note">

一个数字人形象可以有多个数字人音色。
</Note>

## 接口原型

- 请求方法：POST
- 中国大陆正式环境请求地址：https://aigc-api.zegotech.cn/?Action=DescribeTimbreByMetaHuman
- 传输协议：HTTPS

- 调用频率限制：10 次/秒


## 请求参数

以下请求参数列表仅列出了接口请求参数，公共参数列表请参考 [调用方式 - 公共请求参数\|_blank](#18534#4_1)。

| 参数        | 类型   | 是否必选 | 描述            |
| :---------- | :----- | :------- | :-------------- |
| MetaHumanId | String | 否      | 数字人形象 ID，通过 [查询数字人模特列表\|_blank](!DescribeMetaHumanModel) 获得。（若不填写，将返回全量可用公共音色） |

## 请求示例

- 请求 URL  

    ```
    https://aigc-api.zegotech.cn/?Action=DescribeTimbreByMetaHuman
    &AppId=1234567890
    &SignatureNonce=15215528852396
    &Timestamp=1234567890
    &Signature=7a2c0f11145fb760d607a07b54825013
    &SignatureVersion=2.0
    ```

- 请求消息体

    ```json
    {
        "MetaHumanId": "your_metahuman_id"
    }
    ```
