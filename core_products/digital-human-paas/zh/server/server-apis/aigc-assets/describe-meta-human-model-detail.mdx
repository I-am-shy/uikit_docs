

# 查询数字人模特详情

---

## 描述

调用本接口，传入数字人模特 ID，即可查询该模特下所有数字人形象的详情。

<Note title="Note">

一个数字人模特可以有多个数字人形象。
</Note>

## 接口原型

- 请求方法：POST
- 中国大陆正式环境请求地址：https://aigc-api.zegotech.cn/?Action=DescribeMetaHumanModelDetail
- 传输协议：HTTPS
- 调用频率限制：10 次/秒

## 请求参数

以下请求参数列表仅列出了接口请求参数，公共参数列表请参考 [调用方式 - 公共请求参数\|_blank](#18534#4_1)。

| 参数             | 类型   | 是否必选 | 描述            |
| :--------------- | :----- | :------- | :-------------- |
| MetaHumanModelId | String | 是       | 数字人模特 ID，通过 [查询数字人模特列表\|_blank](!DescribeMetaHumanModel) 获得。 |

## 请求示例

- 请求 URL  

    ```
    https://aigc-api.zegotech.cn/?Action=DescribeMetaHumanModelDetail
    &AppId=1234567890
    &SignatureNonce=15215528852396
    &Timestamp=1234567890
    &Signature=7a2c0f11145fb760d607a07b54825013
    &SignatureVersion=2.0
    ```

- 请求消息体

    ```json
    {
        "MetaHumanModelId": "your_metahumanmodel_id"
    }
    ```
