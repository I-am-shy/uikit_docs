

# 查询视频流任务状态

---

## 描述

调用本接口查询一个数字人视频流任务的详情。


## 接口原型

- 请求方法：POST
- 中国大陆正式环境请求地址：https://aigc-api.zegotech.cn/?Action=DescribeMetaHumanLive
- 传输协议：HTTPS
- 调用频率限制：10 次/秒


## 请求参数

以下请求参数列表仅列出了接口请求参数，公共参数列表请参考 [调用方式 - 公共请求参数\|_blank](#18534#4_1)。

| 参数   | 类型   | 是否必选 | 描述                                       |
| :----- | :----- | :------- | :----------------------------------------- |
| TaskId | String | 是       | 数字人视频流任务 ID，[创建数字人视频流任务\|_blank](!CreateMetaHumanLive) 后获取。 |

## 请求示例


- 请求 URL  

    ```
    https://aigc-api.zegotech.cn/?Action=DescribeMetaHumanLive
    &AppId=1234567890
    &SignatureNonce=15215528852396
    &Timestamp=1234567890
    &Signature=7a2c0f11145fb760d607a07b54825013
    &SignatureVersion=2.0
    ```

- 请求消息体  

    ```json
    {
        "TaskId": "your_task_id"
    }
    ```
