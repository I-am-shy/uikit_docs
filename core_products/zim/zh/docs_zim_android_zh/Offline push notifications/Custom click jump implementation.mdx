# 自定义点击跳转实现

## 前提条件

在自定义点击跳转实现之前，请确保：

- 已集成 ZPNs SDK，详情请参考 [实现离线推送](./Implement%20offline%20push%20notification.mdx)。
- 已在控制台配置离线推送证书。


## 实现流程

收到离线推送后，通知栏会显示推送信息（如下图所示）
<img src="/Users/wuyaolin/Downloads/Image_316112594921225.jpg"/>

1、请联系技术支持配置开启自定义点击跳转功能。

2、请在自定义的 ZPNsMessageReceiver 广播类派生类中（参考实现离线推送->使用 ZPNs SDK 注册离线推送功能）,重写 onNotificationClicked 方法。
```java
protected void onNotificationClicked(Context context, ZPNsMessage message) {
        Log.e("MyZPNsReceiver", "onNotificationClicked message:" + message.toString());
        // 获取透传消息，可以根据此处的信息跳转到指定页面
        String payload = message.getPayload();
        //以下为伪代码，通过 Intent 跳转到一个虚构的空白 Activity
        Intent intent = new Intent(context, BlankActivity.class);
        intent.putExtra("EXTRA_MESSAGE",payload);
        intent.setFlags(Intent.FLAG_ACTIVITY_NEW_TASK | Intent.FLAG_ACTIVITY_CLEAR_TOP);
        context.startActivity(intent);
    }
```
3、收到推送消息后点击该消息，ZPNs 将会回调 onNotificationClicked 事件。
