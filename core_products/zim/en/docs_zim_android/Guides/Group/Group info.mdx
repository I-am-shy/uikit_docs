# Manage group info

- - -


ZEGOCLOUD's In-app Chat (the ZIM SDK) provides the capability of group information management, allowing you to get or modify group info, such as group name, nickname in the group, group avatar, group notice, and more.


## Prerequisites 

- **Create a project** in [ZEGOCLOUD Console](https://console.zegocloud.com). The ZIM service permission is not enabled by default. Before using it, please activate the ZIM service by yourself in [ZEGOCLOUD Console](https://console.zegocloud.com) (for details, please refer to [Project Management - In-app Chat ](#15174)), if you cannot activate the ZIM service, please contact ZEGOCLOUD technical support to activate it.
- Integrate the ZIM SDK. For details, see the **Integrate the SDK** chapter of [Send and receive messages](!Implementation_of_sending_and_receiving).


<Warning title="Warning">

- Before you use this feature, you must join a group first. To join a group, refer to the Chapter **Join a group** of [Manage groups](!GroupManagement).
- All in-group members (group owner included) can use this feature.
</Warning>

![/Pics/In_app_Chat/14535.jpeg](//doc.oa.zego.im/Pics/In_app_Chat/14535.jpeg)


## Get group info

To know the detailed info of the group you joined, call the {getPlatformData(props,queryGroupInfoByGroupIDMap)} method.

You will receive the group info through the callback {getPlatformData(props,ZIMGroupInfoQueriedCallbackMap)} after the query succeeds.


:::if{props.platform=undefined}
<CodeGroup>
```java SampleCode
// Get the group info, such as group name, nicknames, group notice.
zim.queryGroupInfo(group_id, new ZIMGroupInfoQueriedCallback() {
    @Override
    public void onGroupInfoQueried(ZIMGroupInfo groupInfo, ZIMError errorInfo) {
        // Get the result of querying group information through errorInfo.
    }
});
```
</CodeGroup>

:::
:::if{props.platform="iOS"}
<CodeGroup>
```objc SampleCode
// Get the group info, such as group name, nicknames, group notice.
[zim queryGroupInfoByGroupID:groupID callback:^(ZIMGroupInfo * _Nonnull groupInfo, ZIMError * _Nonnull errorInfo) {
        //Implement the event callback for query the group info, such as group name, nicknames, group notice here. 
}];
```
</CodeGroup>
:::
:::if{props.platform="mac"}
<CodeGroup>
```objc SampleCode

}];
```
</CodeGroup>
:::

## Modify the group name

To modify the group name after joining a group, call the {getPlatformData(props,updateGroupNameMap)} method.

All group members receive a notification through the callback {getPlatformData(props,onGroupNameUpdatedMap)} after the group name is modified successfully.


:::if{props.platform=undefined}
<CodeGroup>
```java SampleCode
// Modify the group name.
// groupName is a string with a maximum of 64 bytes, with no special character restrictions.
zim.updateGroupName("new_group_name", group_id, new ZIMGroupNameUpdatedCallback() {
    @Override
    public void onGroupNameUpdated(ZIMError errorInfo) {
    }
});
```
</CodeGroup>

## Modify the group avatar

After a successful modification, all group members will receive a notification through {getPlatformData(props,onGroupAvatarUrlUpdatedMap)}.
