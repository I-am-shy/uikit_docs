# 图片数字人制作指南

- - -

本文介绍如何制作符合规范的图片数字人形象素材。

## 形象图片素材制作

### 制作方法

推荐通过相机拍摄、或软件制作生成图片素材。

### 图片素材要求

制作图片素材时，请参考以下要求：

| 类别           | 说明                                                                                                                                         |
| -------------- | -------------------------------------------------------------------------------------------------------------------------------------------- |
| 图片尺寸       | <ul><li>推荐 16:9 竖屏（或 9:16 比例），脸部清晰，分辨率要求 4K（不得低于 720P）</li><li>  需要彩色照片，暂不支持黑白</li></ul>                                           |
| 人物角度&图片占比 |  <ul><li>推荐正脸正身，侧脸尽量与屏幕角度在 30° 以下</li><li>  支持半身照、全身照（最好是上半身 1/2 ~ 2/3，人物在图片正中间且无其他遮挡）</li></ul>                                                                  |
| 人物风格       |  <ul><li>支持真人形象</li><li>  五官清晰、轮廓清晰或立体的二次元形象</li></ul>                                                                                      |
| 人物动作表情   |  <ul><li>自然站立/坐立，不要有抬手、曲臂、弯腰等大幅度动作</li><li>表情自然，不做夸张表情（如大笑、皱眉等）</li><li>嘴巴闭上不要露牙齿</li></ul>                      |
| 人物细节要求   |  <ul><li>五官清晰不能被遮挡</li><li>头部不要有帽子、夸张发饰等加高头部的物件或配饰，不要有过多碎发或发丝</li><li>脸部和脖子附近不能有遮挡物或其他装饰物（如耳饰、碎发、手等）</li></ul>                                                 |
| 背景           |  <ul><li>以纯色背景为佳</li><li>背景和人物需要有明显区分度</li><li>不支持透明度背景</li></ul>                                                                         |
| 合规要求       |  <ul><li>不能有大量露肤</li></ul>                                                                                                        |

### 素材最佳实践

为了让数字人的形象符合预期，图片中人物建议满足以下要求：

1. **有动作要求：** 推荐露出臀部及以上的上半身，双手自然下垂。
