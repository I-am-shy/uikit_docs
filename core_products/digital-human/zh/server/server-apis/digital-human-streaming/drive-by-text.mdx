# 文本驱动数字人

---

##  描述

通过本接口，您可以使数字人播报指定的输入文本。

## 请求参数

以下请求参数列表仅列出了接口请求参数和部分公共参数，完整公共参数列表请参考 [调用方式 - 公共请求参数](./../Accessing%20Server%20APIs.mdx#公共请求参数)。

| 参数               | 类型   | 是否必选 | 描述             |
|------------------|------|------|-------------------------|
| TaskId      | String | 是    | 数字人实时视频流任务 ID。 通过 [创建数字人实时视频流任务](./Create%20Digital%20Human%20Stream%20Task.mdx) 接口的响应参数获取。    |
| Text      | String | 是    | 播报文本内容。不超过 1800 个字。    |
| InterruptMode      | Number | 否    | 打断模式：<br/>0：排队等待前面的播报任务完成后，再执行本播报任务。<br/>1：中断当前播报任务，立即执行本播报任务。 <br/>如果不填，取值为 0。   |
| TTSConfig  | Object | 是    | TTS 相关配置。 本参数结构，请参考 [TTSConfig](#TTSConfig)。           |

### TTSConfig
| 参数               | 类型   | 是否必选 | 描述             |
| ----------------|------|------|-------------------------|
| TimbreId | String | 是    | 音色 ID。通过 [获取音色列表](./Get%20TTS%20Timbre%20List.mdx) 接口的响应参数获取。 |
| SpeechRate | Number | 否    | 语速。取值范围为 [-500, 500]，如果不填，取值为 0。数值越大，语速越快。 |
| PitchRate | Number | 否 | 音调。取值范围为 [-500, 500]，如果不填，取值为 0。数值越大，音调越高。 |
| Volume | Number | 否 | 音量。取值范围为 [1, 100]，如果不填，取值为 50。数值越大，声音越大。 |

## 请求示例

- 请求地址 URL：

    ```https
    https://aigc-digitalhuman-api.zegotech.cn?Action=DriveByText
    &<公共请求参数>
    ```

- 请求消息体：

    ```json
    {
        "TaskId": "f06d1f5d-c0d1-4845-9a2c-f0a45ee037fd",
        "Text": "针对贵报近日关于美国对华加征高达104%关税的报道，我愿再次表达中国政府的严正立场。001美方不顾国际社会普遍反对，以荒唐的借口对所有贸易伙伴征收所谓“对等关税”，是典型的单边主义、保护主义、经济霸凌，严重侵犯各国正当权益，002严重损害以规则为基础的多边贸易体制，严重冲击全球经济秩序。美方更悍然对中国施加高达104%的关税，是赤裸裸的恐吓、讹诈行径，中方对此决不接受。",
        "TTSConfig": {
            "Volume": 51,
            "SpeechRate": 51,
            "PitchRate": 51,
            "TimbreId": "d4bc0764-c074-44fd-b43e-46acdf87fcd9"
        },
        "InterruptMode": 1
    }
    ```

## 响应参数

| 参数 | 类型 | 描述 |
|------|------|------|
| Code | Number | 返回码。0 表示成功，其他值表示失败。如需了解错误码及响应处理建议，请参考 [返回码](./../Return%20codes.mdx)。 |
| Message | String | 请求结果的说明信息。 |
| RequestId | String | 请求 ID。 |
| └DriveId      | String       | 数字人实时视频流驱动任务 ID。 |

## 响应示例

```json
{
    "Code": 0,
    "Message": "success",
    "RequestId": "e6f95291-7053-4c87-9bd0-b4b8c56aabfd",
    "Data": {
        "DriveId": "ae8c4d88-44fe-469a-aedf-df06849d0fdc"
    }
}
```
