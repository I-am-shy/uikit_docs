

# API 概览

---

数字人能力平台服务提供以下相关 API 接口及回调，可参考 [调用方式](./accessing-server-apis.mdx) 调用以下接口。

## 数字人管理

|接口名称|接口功能| 调用频率限制|
|-|-|-|
|[查询数字人列表](./digital-human-management/get-digital-human-list.mdx)| 查询可用的数字人列表。| 10 次/秒 |
|[查询数字人信息](./digital-human-management/get-digital-human-info.mdx)| 查询数字人信息。| 10 次/秒 |
|[查询音色列表](./digital-human-management/get-timbre-list.mdx)| 查询数字人可用的音色。| 10 次/秒 |

|-|-|-|
