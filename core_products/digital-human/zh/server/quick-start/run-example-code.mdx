# 跑通示例源码

我们提供了两种场景的示例源码，分别是数字人实时播报和数字人 AI 互动对话。

示例源码地址：[GitHub](https://github.com/ZEGOCLOUD/digital-human-quick-start-example)

请先下载示例源码以便后续运行示例代码：
```bash
git clone https://github.com/ZEGOCLOUD/digital-human-quick-start-example.git
```

## 数字人实时播报

<Video src="https://doc-media.zego.im/core_products/digital-human/zh/server/quick-start/real-time-broadcasting.mp4" />

[数字人实时播报](https://github.com/ZEGOCLOUD/digital-human-quick-start-example/tree/main/digital-human-real-time-broadcasting-scenario)示例代码包含了服务端及客户端示例代码。请按以下说明运行示例代码：

<Tabs>
<Tab title="Server">

<Note title="注意">
请先运行服务端后再运行客户端。
</Note>

<Steps titleSize="p">
<Step title="进入数字人实时播报示例代码的 server 目录">
```bash
cd digital-human-quick-start-example/digital-human-real-time-broadcasting-scenario/server
```

</Step>
<Step title="安装依赖">
```bash
npm install
```
</Step>
<Step title="配置环境变量">
```bash
cp .env.example .env
```

编辑 `.env` 文件，填入您的 **AppID** 和 **ServerSecret**。

```env
APP_ID=您的_AppID
SERVER_SECRET=您的_ServerSecret
TOKEN_EXPIRE_SECONDS=3600
```
</Step>
<Step title="启动服务器">
```bash
npm run dev
```
</Step>
<Step title="打开播报内容编排页面">
打开浏览器，访问 `http://localhost:3000`，您将看到播报内容编排页面。选择数字人形象和音色，配置播报内容，点击启动播报任务按钮，您将可以通过客户端展示数字人播报。
</Step>
</Steps>

</Tab>
<Tab title="Android">

<Steps titleSize="p">
<Step title="打开 Android Studio 并运行项目">
打开 Android Studio，加载示例代码中的 Android 项目。

编辑 `app/src/main/java/com/example/digitalhumanquickstartdemo/config/Config.kt` 配置文件：

```kotlin
object Config {
    const val APP_ID: Long = 1234567890L  // 替换为你的 AppID
    const val API_BASE_URL = "http://10.0.2.2:3000"  // 模拟器使用 10.0.2.2，真机使用电脑局域网 IP
}
```

点击运行按钮，安装并启动应用。
</Step>
<Step title="选择播报任务并播放">
应用启动后，将自动获取播报列表并选择第一个播报任务进行播报。您此时应该看到数字人播报画面。
</Step>
</Steps>

</Tab>
<Tab title="iOS">

<Steps titleSize="p">
<Step title="安装依赖并打开项目">
```bash
cd digital-human-quick-start-example/digital-human-real-time-broadcasting-scenario/ios-oc
pod install
```

打开 `ZegoDigitalHumanQuickStart.xcworkspace`（注意是 xcworkspace 文件）。
</Step>
<Step title="配置 AppID 和后台地址">
编辑 `ZegoDigitalHumanQuickStart/Config.m`：

```objc
@implementation Config

+ (NSUInteger)APP_ID {
    return 1234567890;  // 替换为你的 AppID
}

+ (NSString *)API_BASE_URL {
    return @"http://localhost:3000";  // 模拟器使用 localhost，真机使用电脑局域网 IP
}

@end
```
</Step>
<Step title="运行项目">
在 Xcode 中选择目标设备，点击运行按钮。
</Step>
<Step title="选择播报任务并播放">
应用启动后，将自动获取播报列表并选择第一个播报任务进行播报。您此时应该看到数字人播报画面。
</Step>
</Steps>

## 数字人 AI 互动对话

<Video src="https://doc-media.zego.im/core_products/digital-human/zh/server/quick-start/interactive-chat.mp4" />

[数字人 AI 互动对话](https://github.com/ZEGOCLOUD/digital-human-quick-start-example/tree/main/digital-human-interactive-chat-scenario) 示例代码包含了服务端及客户端示例代码。请按以下说明运行示例代码：

<Tabs>
<Tab title="Server">

<Note title="注意">
请先运行服务端后再运行客户端。
</Note>

<Steps titleSize="p">
<Step title="进入数字人 AI 互动对话示例代码的 server 目录">
```bash
cd digital-human-quick-start-example/digital-human-interactive-chat-scenario/server
```

</Step>
<Step title="安装依赖">
```bash
npm install
```
</Step>
<Step title="配置环境变量">
```bash
cp .env.example .env
```

编辑 `.env` 文件，填入您的 **AppID**、**ServerSecret** 和 **DigitalHumanID**。

```env
APP_ID=您的_AppID
SERVER_SECRET=您的_ServerSecret
DIGITAL_HUMAN_ID=您的_DigitalHumanID
```
<Note title="注意">
DIGITAL_HUMAN_ID 可通过调用数字人服务端 API `GetDigitalHumanList` 接口获取。或者在播报内容编排页面直接复制。
</Note>

</Step>
<Step title="启动服务器">
```bash
npm run dev
```
</Step>
</Steps>
