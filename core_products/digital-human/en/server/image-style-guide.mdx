# Image-based Digital Human Production Guide

---

This document describes how to create compliant image-based digital human assets.

## Image Asset Production

### Production Methods

It is recommended to use a camera to shoot or use software to generate image assets.

### Image Asset Requirements

Please refer to the following requirements when creating image assets:

| Category                | Description                                                                                                                                                                                                 |
|-------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Image Size              | <ul><li>Recommended 16:9 portrait (or 9:16 ratio), clear face, 4K resolution required (not lower than 1080P)</li><li>Color photos required, black and white not supported</li></ul>                          |
| Face Angle & Proportion | <ul><li>Front-facing, full body recommended; side face angle should be less than 30Â° to the screen</li><li>Half-body (best) and full-body photos are supported</li></ul>                                    |
| Style                   | <ul><li>Supports real person images</li><li>Clear facial features, clear or three-dimensional anime-style images</li></ul>                                            |
| Pose & Expression       | <ul><li>Natural standing/sitting, no large movements such as raised hands, bent arms, or bending over</li><li>Natural expression, no exaggerated expressions (e.g., big smile, frown, etc.)</li><li>Mouth closed, no teeth showing</li></ul> |
| Details                 | <ul><li>Facial features must be clear and not blocked</li><li>No hats, exaggerated headwear, or accessories that increase head height; avoid excessive stray hair</li></ul>                                 |
| Background              | <ul><li>Solid color background is preferred</li><li>Clear distinction between background and subject</li><li>Transparent backgrounds are not supported</li></ul>                                            |
| Compliance              | <ul><li>Adult images required</li><li>No excessive skin exposure</li></ul>                                                                                           |
