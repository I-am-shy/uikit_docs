# Image-based Digital Human Production Guide

---

This document describes how to create compliant image-based digital human assets.

## Image Asset Production

### Production Methods

It is recommended to use a camera to shoot or use software to generate image assets.

### Image Asset Requirements

Please refer to the following requirements when creating image assets:

| Category                | Description                                                                                                                                                                                                                  |
|-------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Image Size              | <ul><li>Recommended 16:9 portrait (or 9:16 ratio), clear face, resolution required to be 4K (not lower than 720P)</li><li>Color photos required, black and white not supported</li></ul>                                     |
| Character Angle & Ratio | <ul><li>Frontal face and body recommended, side face should be within 30Â° of the screen angle</li><li>Half-body or full-body photos supported (preferably upper body 1/2 ~ 2/3, character centered with no other obstructions)</li></ul> |
| Character Style         | <ul><li>Supports real person images</li><li>Clear facial features, clear or three-dimensional 2D anime style</li></ul>                                                                |
| Character Pose & Expression | <ul><li>Natural standing/sitting, avoid large movements such as raising hands, bending arms, or bending waist</li><li>Natural expression, avoid exaggerated expressions (such as laughing, frowning, etc.)</li><li>Mouth closed, no visible teeth</li></ul> |
| Character Detail Requirements | <ul><li>Facial features must not be obscured</li><li>No hats, exaggerated hair accessories, or items/accessories that increase head height; avoid excessive stray hair</li><li>No obstructions or other accessories (such as earrings, stray hair, hands, etc.) near the face and neck</li></ul> |
| Background              | <ul><li>Solid color background is preferred</li><li>Clear distinction between background and character</li><li>Transparent backgrounds are not supported</li></ul>                                                          |
| Compliance Requirements | <ul><li>No excessive skin exposure</li></ul>                                                                                                                                                                                |

### Best Practices

To ensure the digital human's appearance meets expectations, it is recommended that the person in the image follows these guidelines:

1. **For action-required scenarios:** It is recommended to show the upper body from the hips and above, with both arms naturally hanging down.
