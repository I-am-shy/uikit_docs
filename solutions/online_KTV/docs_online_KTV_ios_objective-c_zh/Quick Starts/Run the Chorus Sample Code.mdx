# 合唱示例 Demo 运行指引

- - -


<Card title="示例源码" href="https://storage.zego.im/downloads/KTV_demo/KTVChorusDemo_Express_iOS.zip" target="_blank">点击获得完整代码。</Card>



## 1 准备环境

在运行示例 Demo 前，请确保开发环境满足以下要求：

- Xcode 15.0 或以上版本。
- iOS 12.0 或以上版本且支持音视频的 iOS 设备。
- iOS 设备已经连接到 Internet。

## 2 前提条件

- 已参考 [下载\|_blank](!online_ktv_download)，下载 ZEGO Express SDK（含版权音乐功能）。
- 已在 [ZEGO 控制台\|_blank](https://console.zego.im) 创建项目，并申请有效的 AppID 及 AppSign，详情请参考 [控制台 - 项目管理\|_blank](#12107) 中的“项目信息”。
- 已联系 ZEGO 商务人员为 AppID 开通版权音乐服务。

## 3 运行项目

a. 添加 SDK

   由于示例 Demo 里没有 Express SDK（含版权音乐功能），你需要参考 [集成 SDK\|_blank](!ExpressVideoSDK-Integration/SDK_Integration)，将下载好的 SDK 添加到项目中。SDK 存放路径：KTVChorusDemo_iOS -> KTVDemo -> libs。

![/Pics/ktv/sdk.png](https://storage.zego.im/sdk-doc/Pics/ktv/sdk.png)

b. 打开项目

   双击选择项目中 KTVDemo.xcworkspace 文件，打开项目。

![/Pics/ktv/dir.png](https://storage.zego.im/sdk-doc/Pics/ktv/dir.png)

c. 设置开发者证书

   打开项目 Signing & Capabilities 配置，设置开发者证书，选择自己的开发者证书即可。

![/Pics/ktv/team.png](https://storage.zego.im/sdk-doc/Pics/ktv/team.png)

d. 设置 AppID

请搜索 ZEGOConfig 文件，配置 AppID 和 AppSign，将从控制台获取的 AppID 和 AppSign 填到如图所示的位置。

![/Pics/ktv/appid.png](https://storage.zego.im/sdk-doc/Pics/ktv/appid.png)

e. 运行

<Note title="Note">
项目包含版权音乐模块的代码，你需要为 AppID 开通版权音乐服务才能正常体验 Demo 功能。
</Note>

![/Pics/ktv/iOS_run.png](https://storage.zego.im/sdk-doc/Pics/ktv/iOS_run.png)

## 4 示例 Demo App 点歌播放流程

a. 进入 App 主页后，点击“点歌”按钮，进入“点歌”界面；

b. 选择一首歌曲，点击右侧“点歌”按钮，进行下载；

c. 歌曲下载完成后，"已点列表"更新；

d. 在已点列表中"选择"演唱歌曲；

e. 返回至主页，点击页面左下角“播放”，开始播放歌曲。
