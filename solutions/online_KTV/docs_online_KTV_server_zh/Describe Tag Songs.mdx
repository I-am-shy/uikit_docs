# 根据标签获取歌曲

- - -

## 1 描述

本接口调用需通过 [获取标签列表](/online_ktv_server/describe-tags) 中的标签 ID，可获取对应的歌曲 ID、歌曲名、歌曲专辑、以及版权相关信息等。

<Note title="Note">

相比原有 [歌单](#13719) 接口，本接口歌曲资源更丰富，且支持动态更新。因此，ZEGO 推荐开发者使用本接口获得歌曲列表相关内容。
</Note>

## 2 接口原型

- 请求方法：GET
- 请求地址：https://ktv-api.zego.im/?Action=DescribeTagSongs
- 传输协议：HTTPS
- 调用频率限制：200 次/秒

## 4 请求示例

```
https://ktv-api.zego.im/?Action=DescribeTagSongs
&TagId=1107
&Page=1
&Filter[]=1
&Filter[]=3
&<公共请求参数>
```

以上示例表示：根据标签 ID 是 1107，通过过滤选项 1 和 3，筛选出有词曲版权且含有歌词的歌曲，展示列表第一页的内容。
