# 获取歌曲的歌词

- - -

## 1 描述

通过歌曲 ID 获取歌曲的整首歌词。

## 2 接口原型

- 请求方法：GET
- 请求地址：https://ktv-api.zego.im/?Action=DescribeSongLyric
- 传输协议：HTTPS
- 调用频率限制：200 次/秒



## 3 请求参数

以下请求参数列表仅列出了接口请求参数和部分公共参数，完整公共参数列表请参考 [调用方式 - 公共请求参数\|_blank](!Copyright_Music_Backstage/Call_method#2_1)。

## 4 请求示例

```
https://ktv-api.zego.im/?Action=DescribeSongLyric
&SongId=125282604
&<公共请求参数>
```
