# 获取歌曲附加信息

- - -

## 1 描述

通过获取歌曲附加信息，可以知道该歌曲的节拍数、节拍类型、以及所属流派信息。

@@@KTV_vendorID_note@@@

## 2 接口原型

- 请求方法：GET
- 请求地址：https://ktv-api.zego.im/?Action=DescribeSongExtra
- 传输协议：HTTPS
- 调用频率限制：200 次/秒



## 3 请求参数

以下请求参数列表仅列出了接口请求参数和部分公共参数，完整公共参数列表请参考 [调用方式 - 公共请求参数\|_blank](!Copyright_Music_Backstage/Call_method#2_1)。

## 4 请求示例

```
https://ktv-api.zego.im/?Action=DescribeSongExtra
&SongId=125282604
&<公共请求参数>
```
