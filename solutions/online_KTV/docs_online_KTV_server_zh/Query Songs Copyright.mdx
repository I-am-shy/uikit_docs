# 查询歌曲的版权信息

- - -


## 1 描述

查询歌曲的版权信息，包含词曲伴奏版权、录音版权、及歌曲渠道。

@@@KTV_vendorID_note@@@

## 2 接口原型

- 请求方法：GET
- 请求地址：https://ktv-api.zego.im/?Action=QuerySongsCopyright
- 传输协议：HTTPS
- 调用频率限制：200 次/秒



## 3 请求参数

以下请求参数列表仅列出了接口请求参数和部分公共参数，完整公共参数列表请参考 [调用方式 - 公共请求参数](!Copyright_Music_Backstage/Call_method#2_1)。

@@@KTVServerAPIReminder@@@

<table>
  
  <tbody><tr>
    <th>参数</th>
    <th>类型</th>
    <th>是否必选</th>
    <th>描述</th>
  </tr>
  <tr>
    <td>SongsId[]</td>
    <td>Array of String</td>
    <td>是</td>
    <td>歌曲 ID 列表，最多支持 20 个。</td>
  </tr>
</tbody></table>



## 4 请求示例

```
https://ktv-api.zego.im/?Action=QuerySongsCopyright
&SongsId[]=125282604
&SongsId[]=125282605
&<公共请求参数>
```
