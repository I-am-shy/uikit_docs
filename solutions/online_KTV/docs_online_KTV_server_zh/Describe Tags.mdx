# 获取标签列表

- - -

## 1 描述

通过获取标签列表，可以知道歌曲资源的标签列表、标签分组名称、标签 ID、标签名称等信息。

<Note title="Note">

1. 相比原有 [歌单\|_blank](#13719) 接口，本接口歌曲资源更丰富，且支持动态更新。因此，ZEGO 推荐开发者使用本接口获得歌曲列表相关内容。
2. 调用 [根据标签获取歌曲\|_blank](!Get_songs_based_tags) 接口需要根据标签 ID 获取对应的歌曲资源。
</Note>


## 2 接口原型

- 请求方法：GET
- 请求地址：https://ktv-api.zego.im/?Action=DescribeTags
- 传输协议：HTTPS
- 调用频率限制：200 次/秒

## 3 请求参数

本接口仅需要传入公共参数即可调用，完整公共参数列表请参考 [调用方式 - 公共请求参数\|_blank](!Copyright_Music_Backstage/Call_method#2_1)。

@@@KTVServerAPIReminder@@@
