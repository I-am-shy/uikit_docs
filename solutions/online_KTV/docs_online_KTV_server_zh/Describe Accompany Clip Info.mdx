# 获取伴奏高潮时间点

- - -

## 1 描述

通过获取伴奏高潮时间点，可以标示出一首伴奏中高潮片段的时间节点。

## 2 接口原型

- 请求方法：GET
- 请求地址：https://ktv-api.zego.im/?Action=DescribeAccompanyClipInfo
- 传输协议：HTTPS
- 调用频率限制：200 次/秒



## 3 请求参数

以下请求参数列表仅列出了接口请求参数和部分公共参数，完整公共参数列表请参考 [调用方式 - 公共请求参数](!Copyright_Music_Backstage/Call_method#2_1)。
