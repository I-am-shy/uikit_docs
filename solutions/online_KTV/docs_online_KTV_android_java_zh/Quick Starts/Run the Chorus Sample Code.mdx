- - -

<Card title="示例源码" href="https://storage.zego.im/downloads/KTV_demo/KTVChorusDemo_Android.zip" target="_blank">点击获得完整代码。</Card>



## 1 准备环境

在运行示例 Demo 前，请确保开发环境满足以下要求：

- Android Studio 2020.3.1 或以上版本。
- Android SDK 30、Android SDK Platform-Tools 30.x.x 或以上版本。
- Android 4.4 或以上版本，且支持音视频的 Android 设备（仅支持真机），并开启“允许调试”选项。
- Android 设备、macOS 开发电脑已经连接到 Internet。

## 2 前提条件

- 已参考 [下载\|_blank](!online_ktv_download)，下载 ZEGO Express SDK（含版权音乐功能）。
- 已在 [ZEGO 控制台\|_blank](https://console.zego.im) 创建项目，并申请有效的 AppID 及 AppSign，详情请参考 [控制台 - 项目管理\|_blank](#12107) 中的“项目信息”。
- 已联系 ZEGO 商务人员为 AppID 开通版权音乐服务。

## 3 运行项目

a. 添加 SDK

   由于示例 Demo 里没有 ZEGO Express SDK（含版权音乐功能），你需要参考 [集成 SDK\|_blank](!ExpressVideoSDK-Integration/SDK_Integration)，将下载好的 SDK 添加到项目中。SDK 存放路径：KTVChorusDemo_Android -> app->libs。

![/Pics/ktv/libs.png](http://doc.oa.zego.im/Pics/ktv/libs.png)

b.  打开项目

选择 KTVChorusDemo_Android 项目在 IDE 中打开。

![/Pics/ktv/as_open.png](http://doc.oa.zego.im/Pics/ktv/as_open.png)

c. 设置 appID

请搜索 ZegoDataCenter 文件，配置 AppID 和 AppSign，将从控制台获取的 AppID 和 AppSign 填到如图所示的位置。

![/Pics/ktv/modify_app_config.png](http://doc.oa.zego.im/Pics/ktv/modify_app_config.png)

d. 运行

<Note title="Note">
项目包含版权音乐模块的代码，你需要为 AppID 开通版权音乐服务才能正常体验 Demo 功能。
</Note>

![/Pics/ktv/android_run.png](http://doc.oa.zego.im/Pics/ktv/android_run.png)

## 4 示例 Demo App 点歌播放流程

a. 进入 App 主页后，点击“点歌”按钮，进入“点歌”界面；

b. 选择一首歌曲，点击右侧“点歌”按钮，进行下载；

c. 歌曲下载完成后，"已点列表"更新；

d. 在已点列表中"选择"演唱歌曲；

e. 返回至主页，点击页面左下角“播放”，开始播放歌曲。
