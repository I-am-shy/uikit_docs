# 全局返回码
---
当出现错误时，可根据下方表格查询对应的错误原因。

| 返回码 | 描述     |
| ------- | ---------- |
| 0 	  | 成功。 	|
| 1       | 异常。       |
| 2       | 重复请求。   |
| 3       | 系统错误。   |
| 4       | 错误的参数。 |
