# 如何在线调试服务端 API

您可以在 ZEGO 服务端 API 文档页面在线调试服务端 API，方便开发者快速测试和验证 API 的功能。
请从 [ZEGO 控制台](https://console.zego.im) 获取 AppId 和 ServerSecret 后填入页面右侧面板的 ZEGO 签名生成器生成 API 必要请求参数，生成后再填写或修改其他参数即可发起请求进行测试。


<Note title="说明">计算签名的操作在客户端完成，不会向服务端发送任何数据。</Note>
<Warning title="注意">
- 请确保 AppId 和 ServerSecret 的准确性
- 生成的签名用于服务端 API 调用的身份验证
- 签名具有时效性，请及时使用。如果接口报错提示签名过期，请重新生成签名。
</Warning>
