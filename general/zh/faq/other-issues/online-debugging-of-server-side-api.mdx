# 如何在线调试服务端 API

## 1. 概述

Zego 提供在线调试服务端 API 的工具，方便开发者快速测试和验证 API 的正确性。

## 2. 使用方法

请从 ZEGO 控制台获取 APPID 和 ServerSecret，然后填入以下输入框后点击生成按钮，即可生成签名。

<SignatureGenerator />

## 3. 注意事项

- 请确保 AppId 和 ServerSecret 的准确性
- 生成的签名用于服务端 API 调用的身份验证
- 签名具有时效性，请及时使用
