# 在 Windows 平台上集成 Express uni-app SDK 后，使用 dev 证书云打包运行 iOS 应用时出错，该如何处理？

- - -
  
## 问题描述

在 Windows 平台上集成 Express uni-app 的 SDK，使用 dev 证书云打包运行 iOS 应用时，可能会出现“解压文件失败”的错误提示：

<img src="https://doc-media.zego.im/sdk-doc/Pics/QuickStart/uniapp_dev_error.png">
  

## 解决方案

- 如果使用 adhoc 证书打包，直接运行无异常。

- 如果使用 dev 证书打包，出现上述错误时：

    请手动安装自定义基座，基座路径：unpackage > debug > iOS_debug.ipa，并右键选择“安装到手机”，如下图所示：

    <img src="https://doc-media.zego.im/sdk-doc/Pics/QuickStart/uniapp_dev_error_solution.png">
    <img src="https://doc-media.zego.im/sdk-doc/Pics/QuickStart/uniapp_dev_error_solution1.png">

    <div class="mk-hint">

    Windows 平台上识别 Apple 设备时，需要提前安装 iTunes 应用。
    </div>
