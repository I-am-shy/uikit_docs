# ZIM 运营数据

---

本文介绍如何在 [ZEGO 控制台](https://console.zego.im) 查看各项 ZIM 运营数据。

## 数据概述

- **前提条件**：当前开通了 ZIM 服务时，控制台才会展示 ZIM 的用量数据。
- **数据范围**：通常情况下只会展示最近 1 年的数据，最长时间范围可以选择 3 个月内的数据。

### 数据分类及说明

`ZIM 运营数据` 菜单包含以下子菜单

<Frame >
  <img src="https://media-resource.spreading.io/docuo/workspace564/27e54a759d23575969552654cb45bf89/cb3bcb8d08.png" />
</Frame>

按照数据类型区分，`ZIM 运营数据`有以下几个分类：

| 数据分类 | 数据名           | 数据描述                         |
| -------- | ---------------- | -------------------------------- |
| 用户     | 活跃用户         | 上一日的活跃用户数               |
| !mu      | 本月DAU峰值      | 本月某一天中日活用户数最大的数量 |
| !mu      | 日活用户国家分布 | 不同国家日活用户数量，按数量排列 |
| 消息     | 单聊消息数       | AppID 下一段时间的消息数          |
| !mu      | 群聊消息数       | AppID 下一段时间的消息数          |
| !mu      | 房间消息数       | AppID 下一段时间的消息数          |
| 房间     | 房间并发峰值     | 截止本月上一日的房间并发的峰值   |
| !mu      | 房间总数         | 截止本月上一日创建的房间总数     |
| !mu      | 房间并发人数峰值 | 截止本月上一日房间并发人数峰值   |

- 消息数据：消息数据包含单聊消息数、群聊消息数和房间消息数。
