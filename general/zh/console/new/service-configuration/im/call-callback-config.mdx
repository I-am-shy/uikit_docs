# ZIM 相关回调配置

- - -

## 概述

在使用 ZIM 时，开发者可以在 [ZEGO 控制台](https://console.zego.im) 设置回调接收地址，对接业务后台，进一步保证有序正常开展业务。有关 ZIM 呼叫邀请回调的具体介绍，请参考即时通讯的 [服务端 API - 回调配置说明](/zim-server/callbacks/callback-configuration-instructions)。

本文介绍如何在 [ZEGO 控制台](https://console.zego.im) 上配置呼叫邀请相关回调。配置前需要在 [ZEGO 控制台](https://console.zego.im) 自助开通即时通讯，详情请参考 [即时通讯 - 开通服务](!Platform-IM)。

## 配置流程

<Steps>

<Step title="">
前往 [ZEGO 控制台](https://console.zego.im)，点击 **“项目管理”**，选择需要配置的项目点击 **“查看”**。选择 **“服务配置 - 即时通讯”**。
<Frame width="auto" height="auto" >
  <img src="https://media-resource.spreading.io/docuo/workspace741/896bc39e2e65b82d5670b01b7c131c30/e24652a7e9.png"/>
</Frame>
</Step>
<Step title="">
向下滚动页面找到 **“ZIM相关回调配置”** 模块，点击 **“编辑”** 打开编辑回调地址弹窗。
<Frame width="512" height="auto" caption=""><img src="https://doc-media.zego.im/sdk-doc/Pics/Consle/pic_1.jpeg" /></Frame>
</Step>
<Step title="">
在弹窗中输入回调地址，完成后单击 **“确定”** 完成配置。
<Frame width="512" height="auto" caption=""><img src="https://doc-media.zego.im/sdk-doc/Pics/Consle/pic_2.jpeg" /></Frame>
</Step>

</Steps>
