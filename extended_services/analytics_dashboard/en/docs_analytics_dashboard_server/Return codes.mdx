# Return codes

- - -

|Return code|Description| Suggested Action |
|---|----|----|
| 0      | Success. | -        |
| 500    | Interface error. | Contact ZEGOCLOUD Technical Support. |
| 10001  | Parameter format error. | Please check the parameters. |
| 10002  | Parameter error or invalid parameter. | Please check the parameters. |
| 30002  | Interface request frequency exceeds the limit.	 | Please confirm the QPS limit for the corresponding interface and reduce the request frequency. |
