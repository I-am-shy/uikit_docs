## Version 1.0.0

**Release Date: 2022-10-28**

**New Features**

ZEGO's audio and video service platform `Prism` has been upgraded to `Analytics Dashboard`, adding 4 major functional modules:

| Feature | Description |
| -- | -- |
| Quality Dashboard | Developers can overview user experience and service status to measure business quality. |
| Operation Dashboard | Developers can understand changes in business scale. |
| Real-time Monitoring | Provides minute-level real-time data for developers to monitor business experience and service status changes, enabling timely adjustments to business operation strategies to ensure service quality. |
| Monitoring Alerts | Developers can set alerts for multiple metrics of audio and video services, customize alert rules with flexible notification methods, receive alert anomalies immediately, and respond promptly to ensure service stability. |
