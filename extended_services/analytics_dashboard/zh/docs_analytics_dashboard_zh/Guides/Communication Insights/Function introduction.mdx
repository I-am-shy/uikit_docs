Query All Running Video Stream Tasks
Description
Invoke this API to query the details of all running video stream tasks.
API Prototype
Request Method: POST
Request URL for Mainland China Production Environment: https://aigc-api.zegotech.cn/?Action=DescribeMetaHumanLiveTasks
Note: I encountered an issue while parsing the provided URL. This problem might be related to the URL itself or the network connection. If you need specific content from this URL, please check the validity of the link and try again.
Transfer Protocol: HTTPS
Rate Limit: 10 requests per second

This API only requires common parameters. For the list of common parameters, please refer to Calling Method - Common Request Parameters.
