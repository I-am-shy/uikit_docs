# API overview

- - -
The Super Board server provides the following Server APIs. 

### Server APIs related to permissions on a whiteboard

|API|Description|
|-|-|
|[Set room permissions](./set-room-permissions.mdx)|Sets default room permissions on a whiteboard. |
|[Set user permissions](./set-user-permissions.mdx)|Sets user permissions on a whiteboard. |
|[Query room permissions](./get-room-permissions.mdx)|Queries room permissions on a whiteboard. |
|[Query user permissions](./get-user-permissions.mdx)|Queries user permissions on a whiteboard. |
