# API overview

- - -
The Super Board server provides the following Server APIs. 

### Server APIs related to permissions on a whiteboard

|API|Description|
|-|-|
|[Set room permissions](./Set%20Room%20Permissions.mdx)|Sets default room permissions on a whiteboard. |
|[Set user permissions](./Set%20User%20Permissions.mdx)|Sets user permissions on a whiteboard. |
|[Query room permissions](./Get%20Room%20Permissions.mdx)|Queries room permissions on a whiteboard. |
|[Query user permissions](./Get%20User%20Permissions.mdx)|Queries user permissions on a whiteboard. |
