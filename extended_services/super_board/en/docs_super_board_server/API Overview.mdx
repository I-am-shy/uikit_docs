# API overview

- - -
The Super Board server provides the following Server APIs. 

### Server APIs related to permissions on a whiteboard

|API|Description|
|-|-|
|[Set room permissions](!ServerAPI/Set_Room_Permissions)|Sets default room permissions on a whiteboard. |
|[Set user permissions](!ServerAPI/Set_User_Permissions)|Sets user permissions on a whiteboard. |
|[Query room permissions](!ServerAPI/Get_Room_Permissions)|Queries room permissions on a whiteboard. |
|[Query user permissions](!ServerAPI/Get_User_Permissions)|Queries user permissions on a whiteboard. |
