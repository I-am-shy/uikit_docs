Here we provide you with a sample application to experience the ZEGOCLOUD's Super Board, and the following describes how to run the sample code.

### Prerequisites
- Xcode 7.0 or later.
- An iOS device or iOS Simulator that is running on iOS 11.0 or later and supports audio and video. We recommend you use a real device.
- The iOS device is connected to the internet.

- Contact the ZEGOCLOUD Technical support to get the AppID and AppSign for SDK integration.


@@@Server_Open_Note@@@

### Run the sample code
1. Open AppStore to download the Xcode.

<Frame width="512" height="auto" caption=""><img src="https://storage.zego.im/sdk-doc/Pics/iOS/GoClass/appstore-xcode.png" /></Frame>

2. In Xcode, select the **File > Open**, open the `ZegoSuperBoardDemo.xcworkspace` file in the extracted sample code package.

<Frame width="512" height="auto" caption=""><img src="https://storage.zego.im/sdk-doc/Pics/iOS/GoClass/xcode-open-file.png" /></Frame>

<Frame width="512" height="auto" caption=""><img src="https://storage.zego.im/sdk-doc/Pics/iOS/WhiteboardExample/whiteboard_demo_dir_workspace.png" /></Frame>

3. In Xcode, select the **Preferences**, click the **Accounts** tab, click the "+" button, then select **Apple ID**, and click **Continue**. And then, enter your Apple ID and Password to log in. 

<Frame width="512" height="auto" caption=""><img src="https://storage.zego.im/sdk-doc/Pics/iOS/GoClass/xcode-account.png" /></Frame> 
	
<Frame width="512" height="auto" caption=""><img src="https://storage.zego.im/sdk-doc/Pics/iOS/GoClass/xcode-login-apple-id.png" /></Frame>

4. Modify the `Bundle Identifier` and `certificate`.

a. In Xcode, Select the **ZegoSuperBoardDemo** project.
<Frame width="512" height="auto" caption=""><img src="https://storage.zego.im/sdk-doc/Pics/iOS/WhiteboardExample/whiteboard_demo_target.png" /></Frame>


b. Select the target project, click the **Signing & Capabilities** tab, change the **Bundle Identifier** configuration and select your own developer certificate in the **Team**.
<Frame width="512" height="auto" caption=""><img src="https://storage.zego.im/sdk-doc/Pics/iOS/WhiteboardExample/whiteboard_team_cert.png" /></Frame>

5. Modify the `KeyCenter.m` file with the AppID and AppSign that you get in the **Prerequisites**.

```objc
static unsigned int kAppID = <#YOUR_APP_ID#>;
static NSString * kAppSign = @"<#YOUR_APP_SIGN#>";
```

6. Connect the iOS device to your computer, and select the iOS device in Xcode.

<Frame width="512" height="auto" caption=""><img src="https://storage.zego.im/sdk-doc/Pics/iOS/WhiteboardExample/whiteboard_demo_generic_arch.png" /></Frame>

<Frame width="512" height="auto" caption=""><img src="https://storage.zego.im/sdk-doc/Pics/iOS/WhiteboardExample/whiteboard_demo_run_device.png" /></Frame>

7. Navigate to the folder where the `Podfile` file locates, and run the `pod install` command in the terminal. 

8. Click the **Build** button to run the sample code and experience the Super Board service.

<Frame width="512" height="auto" caption=""><img src="https://storage.zego.im/sdk-doc/Pics/iOS/WhiteboardExample/whiteboard_demo_run_btn.png" /></Frame>
