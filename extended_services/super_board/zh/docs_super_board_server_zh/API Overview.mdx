# API 概览

- - -

超级白板服务端提供以下相关 API 接口。您可参考 [调测指南](!ServerAPI/Postman) 使用 Postman 快速调试下列接口。

<Warning title="Warning">

1. 本文档 API 接口均为最新服务端 API v2 接口，后续相关功能的新增都会在此更新。为了给开发者带来更好的体验，ZEGO 推荐开发者使用最新 API v2 接口。 
2. 文件转码旧版 API 仅供 **2021-09-10** 前接入的旧用户维护使用。旧版接口文档请参考 [旧版服务端 API\|_blank](#11997)。

</Warning>

### 文件转码相关接口


|接口名称|接口功能|
|-|-|
|[请求文件转码](!ServerAPI/Request_File_Transcoding)|通过本接口创建文件转码任务。|
|[查询文件转码状态](!ServerAPI/Query_File_Transcoding_Status)|通过本接口查询文件转码任务状态。|
|[取消文件转码](!ServerAPI/Cancel_File_Transcoding)|通过本接口取消未开始的文件转码任务。|
|[删除文件](!ServerAPI/Delete_Files)|通过本接口将“转码后文件”或者“源文件以及转码后文件”从 ZEGO 云存储删除。|
|[文件转码状态回调](!ServerAPI/File_Covert_Status_Callback)|通过本接口获取文件转码状态的事件回调。|

### 白板权限相关接口

|接口名称|接口功能|
|-|-|
|[设置房间权限](!ServerAPI/Set_Room_Permissions)|调用本接口用来设置房间的默认白板权限。|
|[设置用户权限](!ServerAPI/Set_User_Permissions)|调用本接口设置用户的白板权限。|
|[查询房间权限](!ServerAPI/Get_Room_Permissions)|调用本接口获取房间的默认白板权限。|
|[查询用户权限](!ServerAPI/Get_User_Permissions)|调用本接口用来获取用户的白板权限。|
