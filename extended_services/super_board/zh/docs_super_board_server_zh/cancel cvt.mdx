# 取消文件转码

- - -

## 1 描述

通过本接口取消未开始的文件转码任务，如果转码任务开始后调用此接口会返回错误。

调用频率限制：

- 正式环境：10 次/秒
- 测试环境：10 次/秒

## 2 接口原型

- 请求方法：POST
- 请求地址：https://docservice-api.zego.im/?Action=StopTranscode
- 传输协议：HTTPS

## 3 请求参数

以下请求参数列表仅列出了接口请求参数，公共参数列表请参考 [调用方式 - 公共请求参数\|_blank](11349#2_1)。
