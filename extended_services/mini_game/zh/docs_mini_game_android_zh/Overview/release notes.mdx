## 1.2.0 版本

**发布日期：2023-09-12**

<p class="hthree">新增功能</p>

| 功能项  | 功能描述 | 相关接口 |
|-------|-------|-------|
|  增加通知开关麦克风的回调 | 在游玩部分游戏（如谁是卧底）时，开发者时常需要根据游玩流程开启/关闭玩家的麦克风。游戏会通过 [onMicStateChange\|_blank](@onMicStateChange) 通知开发者及时调用 RTC SDK 的相关接口，开启/关闭玩家的麦克风。 | [onMicStateChange\|_blank](@onMicStateChange) |
|  增加通知开关扬声器的回调 | 在游玩部分游戏（如谁是卧底）时，开发者时常需要根据游玩流程开启/关闭玩家的扬声器。游戏会通过 [onSpeakerStateChange\|_blank](@onSpeakerStateChange) 通知开发者及时调用 RTC SDK 的相关接口，开启/关闭玩家的扬声器。 | [onSpeakerStateChange\|_blank](@onSpeakerStateChange) |

## 1.1.0 版本

**发布日期：2023-07-20**

<p class="hthree">新增功能</p>

首次发布，开发者仅需集成一个 SDK，即可动态加载多种单人及多人对战休闲小游戏。此外，小游戏平台提供完整、安全的金币管理能力，支持设置语言等功能。
