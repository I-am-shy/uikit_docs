# API 概览

- - -

ZEGO 小游戏平台产品服务端 API 支持全球就近接入、统一的鉴权方式、统一的参数风格和公共错误码，给开发者带来简单快捷的使用体验。

## 弹幕游戏相关

ZEGO 小游戏平台产品服务端提供以下与弹幕游戏相关接口：

| 名称 | 描述 | QPS 限制 |
| -- | -- | -- |
| [DescribeGameLaunchCode\|_blank](!ZegoMiniGameEngine-Get_Activation_Code) | 获取游戏启动码。 | 20 次/秒 |
| [SendDanmakuLikes\|_blank](!ZegoMiniGameEngine-Like) | 点赞。 | 100 次/秒 |
| [SendDanmakuComments\|_blank](!ZegoMiniGameEngine-Comments) | 发送评论。 | 50 次/秒 |
| [SendDanmakuGifts\|_blank](!ZegoMiniGameEngine-Send_Gifts) | 送礼。 | 20 次/秒 |
| [DescribeDanmakuMsgResults\|_blank](!ZegoMiniGameEngine-Check_Gift_Sending_Results) | 查询礼物发送结果。 | 20 次/秒 |

## 回调

ZEGO 小游戏平台产品服务端提供以下回调：

| 回调 | 说明 |
| -- | -- |
| [上报普通小游戏信息\|_blank](!ZegoMiniGameEngine-report_game_info) |在玩家进入普通小游戏、普通小游戏结算、玩家退出普通小游戏时收到具体的事件通知。|
| [上报弹幕游戏信息\|_blank](!ZegoMiniGameEngine-report_game_info_barrage) | 在弹幕游戏开始、弹幕游戏结算时收到具体的事件通知。|
