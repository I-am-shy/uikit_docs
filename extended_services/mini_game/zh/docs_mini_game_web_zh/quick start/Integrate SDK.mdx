## 集成 SDK

### 集成 ZEGO MiniGameEngine SDK

1. 通过 npm 安装 SDK。    

    在您的项目文件夹，打开终端，执行 `npm i zego-mini-game-engine` 命令安装依赖。

    <Note title="Note">

    1. npm 下载包支持 typescript 语言（推荐）。
    2. 如果在 macOS 或 Linux 系统中执行 npm 命令失败，提示 “permission denied”，请在 npm 命令前加上 `sudo` 重新执行即可。
    </Note>


2. 引入 SDK。   

    ```js
    import { ZegoMiniGameEngine } from 'zego-mini-game-engine'
    ```

3. 在您需要挂载游戏的页面中添加容器，并设置 `id`。   

    ```html
    <template>
        <div>
            // 弹幕游戏的容器
            <div id="cloudGame"></div>
        </div>
    </template>
    ```

### 集成 ZEGO Express SDK

在实现主播在浏览器开播弹幕游戏之前，还需要额外集成 ZEGO Express SDK 搭配使用。请参考 [实时音视频 - 集成 SDK\|_blank](#199)。如果您的项目已集成相关 SDK，可忽略此步骤，
