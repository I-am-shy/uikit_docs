# Common error codes

---


| Error code| Description| Solution
|----------|----------|----------
| 0| The execution succeeds.| N/A
| 7000001| A network error occurs.| Check whether the network service is normal.
| 7000002| The user information is invalid.| Check whether the user information is valid.
| 7000003| One or more parameters are invalid.| Check whether the parameters are valid.
| 7000004| The token is invalid or expired.| Check whether the token is valid or generate a new token.
| 7000005| The room ID is invalid.| Check whether the room ID is valid.
| 7000006| The game ID is invalid.| Check whether the game ID is valid.
| 7000007| The request times out.| Try again. If the error code persists, contact ZEGO technical support.
| 7000008| ZEGO MiniGameEngine is already initialized.| Do not initialize ZEGO MiniGameEngine again.
| 7000009| ZEGO MiniGameEngine is not initialized.| Initialize ZEGO MiniGameEngine.
| 7000010| No game container is configured.| Configure a game container.
| 7000011| The game information is empty.| Contact ZEGO technical support.
| 7000012| The game is not running.| Load the game.
| 7000013| The gaming mode is not supported.| Check whether the gaming mode is valid.
| 7000014| The game is not supported in the region.| N/A
| 7000015| The game is already loaded.| Do not reload the game.
| 7100000| An internal conversion error occurs.| Contact ZEGO technical support.
| 7100001| An internal sequence error occurs.| Contact ZEGO technical support.
| 7100002| No game is running.| Check whether the game is running. If the game is running but the SDK still returns the error, contact ZEGO technical support.
| 7100003| The iframe tag does not exist. The iframe tag is passed in when you call the `setGameContainer` operation of the SDK for web.| Check whether the iframe ID is valid.
| 7700000| A network error occurs.| Check whether the network service is normal.
| 7700001| The game fails to be loaded.| You may not have the required game permissions. Contact ZEGO technical support.
| 7700002| The user information fails to be obtained.| Possible causes: <ul><li>You may not have the required game permissions.</li> <li>An error occurs in the game. </li></ul>Contact ZEGO technical support.
| 7700003| The game unexpectedly exits due to an internal error.| Contact ZEGO technical support.
| 7700006| The user is restricted.| In most cases, the user behavior is restricted.
| 7701000-7702999| A resource error occurs in the game.| Contact ZEGO technical support.
| 7999999| An unknown error occurs.| Contact ZEGO technical support.
