# 美妆

- - -
## 功能简介

ZegoEffects SDK 提供美妆功能，包含眼线，眼影，眼睫毛，腮红，口红，美瞳，风格妆等，进行局部上妆，打造出自然精致的妆容，可广泛应用于短视频，直播等场景。


## 前提条件

在使用 ZegoEffects SDK 美颜功能前，请确保：

- 已在项目中集成 ZegoEffects SDK，实现基本的图像处理功能，详情请参考 [快速开始 - 实现图像处理](!Quick_Starts/Solution_Implementation)。
- 导入美妆功能资源 “MakeupResources” 文件，详情请参考 [快速开始 - 导入资源和模型](!Quick_Starts/Import_Resources)。


## 使用步骤

美妆提供了以下功能：眼线、眼影、眼睫毛、腮红、口红、美瞳、风格妆。

各个功能包含的风格有：

- [眼线](!AI_Vision-Common_Features/Makeups#3_1)：自然（Natural）、野猫（Cat Eye）、俏皮（naughty）、心机（innocent）、气质（dignified）。
- [眼影](!AI_Vision-Common_Features/Makeups#3_2)：粉雾海（Pink Mist）、微光蜜（Shimmer Pink）、暖茶棕（Tea Brown）、元气橙（Bright Orange）、摩卡棕（Mocha_Brown）。
- [眼睫毛](!AI_Vision-Common_Features/Makeups#3_3)：自然（Natural）、温柔（Tender）、卷翘（Curl）、纤长（Everlong）、浓密（Thick）。
- [腮红](!AI_Vision-Common_Features/Makeups#3_4)：微醺（Slightly Drunk）、蜜桃（Peach）、奶橘（Milky Orange）、杏粉（Apricot Pink）、甜橙（Sweet Orange）。
- [口红](!AI_Vision-Common_Features/Makeups#3_5)：豆沙粉（Cameo Pink）、甜橘色（Sweet Orange）、铁锈红（Rust Red）、珊瑚色（Coral）、丝绒红（Red Velvet）。
- [美瞳](!AI_Vision-Common_Features/Makeups#3_6)：水光黑（Darknight Black）、星空蓝（Starry Blue）、仙棕绿（Mystery Brown-Green）、异瞳棕（Polar Lights Brown）、可可棕（Chocolate Brown）。
- [风格妆](!AI_Vision-Common_Features/Makeups#3_7)：眼睑下至妆（Vulnerable & Innocent Eyes）、银河眼妆（Milky Eyes）、奶凶（Cutie & Cool）、纯欲（Pure & Sexy）、神颜（Flawless）。

<Warning title="Warning">

- 每个功能的各种风格效果不能叠加，设置新的风格资源路径后，会覆盖之前的效果。
- 眼线、眼影、眼睫毛、腮红、口红、美瞳等功能可以叠加使用，但不能与风格妆功能叠加使用。
</Warning>

开启美妆功能之后，开发者可根据实际的需求来自定义美妆参数。如果未自定义，SDK 将采用默认参数值实现美妆功能。



### 眼线

1. 开启眼线。

    调用 [setEyeliner](/unique-api/ai-effects-sdk/zh/dart_flutter/zego_effects_plugin/ZegoEffectsPlugin/setEyeliner.html) 方法开启/关闭眼线功能，传入对应的眼线名字。调用 [setEyelinerParam](/unique-api/ai-effects-sdk/zh/dart_flutter/zego_effects_plugin/ZegoEffectsPlugin/setEyelinerParam.html) 方法设置眼线强度。

    [ZegoEffectsEyelinerParam](/unique-api/ai-effects-sdk/zh/dart_flutter/zego_effects_defines/ZegoEffectsEyelinerParam-class.html) 对象的眼线强度参数 “intensity” 的取值范围为 [0, 100]。取值越大，效果呈现的程度越大，开发者可以根据需求自定义取值。

    ```dart
    // 开启眼线功能
    // 开启眼线功能，并且传入对应的眼线绝对路径
    ZegoEffectsPlugin.instance.setEyeliner('eyelinerdir_natural');

    // 设置眼线强度，范围 [0, 100]
    ZegoEffectsEyelinerParam param = ZegoEffectsEyelinerParam();
    param.intensity = 100;
    ZegoEffectsPlugin.instance.setEyelinerParam(param);
    ```
   
2. 移除眼线。

    如果需要移除眼线，可将空字符串传入 [setEyeliner](/unique-api/ai-effects-sdk/zh/dart_flutter/zego_effects_plugin/ZegoEffectsPlugin/setEyeliner.html) 方法。

    ```java
    // 移除眼线功能
    ZegoEffectsPlugin.instance.setEyeliner('');
    ```
