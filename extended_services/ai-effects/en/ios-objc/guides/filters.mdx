# Filters

---


The ZegoEffects SDK provides video filters with different styles that can be used in a wide range of use cases, such as live video streaming, short videos, and more.

## Prerequisites

Before you begin to use the SDK's video filters  in your project, make sure you complete the following steps:

1. Integrate the ZegoEffects SDK into your project. For details, see [Integration](./../quick-starts/import-the-sdk.mdx).
2. Implement the basic image processing functionality. For details, see [Implementation](./../quick-starts/implement-basic-image-processing.mdx).

## Implementation steps

The SDK provides video filters with the following styles:
- Natural: Creamy, Brighten, Fresh, Autumn
- Gray: Cool, Night, Film-like
- Dreamy: Sunset, Cozily, Sweet

In general, you can take the following steps to use a filter:

1. Enable the filter by specifying the resource path of the filter.
2. Set the filter parameters as needed. If not specified, the SDK uses the default settings.


## Enable a filter 

To enable a filter, do the following:

1. Call the {getPlatformData2(props,setFilterMap)} method, passing the absolute path of the corresponding filter to the `path` parameter.

2. Call the {getPlatformData2(props,setFilterParamMap)} method to set the filter intensity.

   The value range of the `intensity` property of `ZegoEffectsFilterParam` is  [0, 100]. As this value gets larger, the filter effect gets more intense.

<Warning title="Warning">
You can't superimpose the effects of different filters. Once you apply a new filter, the previous one (if any) will be overridden.
</Warning>

:::if{props.platform=undefined}
```objc
// Enable a filter, passing in the absolute path of the corresponding filter.
[self.effects setFilter:@"/xxx/xxx/Night.bundle"];

// Set the filter intensity. The value range is [0, 100].
ZegoEffectsFilterParam *param = [[ZegoEffectsFilterParam alloc] init];
param.intensity = 100;
[self.effects setFilterParam:param];
```
:::
:::if{props.platform="Android"}
```java
// Enable a filter, passing in the absolute path of the corresponding filter.
mEffects.setFilter("/xxx/xxx/Night.bundle");

// Set the filter intensity. The value range is [0, 100].
ZegoEffectsFilterParam param = new ZegoEffectsFilterParam();
param.intensity = 100;
mEffects.setFilterParam(param);
```
:::
:::if{props.platform="Flutter"}
```dart
// Enable a filter
// 1. Enable the filter, passing in the absolute path of the corresponding filter.
ZegoEffectsPlugin.instance.setFilter('Night');

// Set the filter intensity. The value range is [0, 100].
ZegoEffectsFilterParam param = ZegoEffectsFilterParam();
param.intensity = 100;
ZegoEffectsPlugin.instance.setFilterParam(param);
```
:::
:::if{props.platform="RN"}
```javascript
// Enable a filter
// 1. Enable the filter, passing in the absolute path of the corresponding filter.
mEffects.setFilter("/xxx/xxx/Night.bundle");

// Set the filter intensity. The value range is [0, 100].
mEffects.setFilterParam({ intensity: 100 });
```
:::

## Disable the filter 

To disable the filter, call the `setFilter` method with the `path` parameter set to `nil`.

If you need to remove the filter, you can pass {getPlatformData2(props,nilMap)} to the {getPlatformData2(props,setFilterMap)} method.

:::if{props.platform=undefined}
```objc
// Disable the filter.
[self.effects setFilter:nil];
```
:::
:::if{props.platform="Android"}
```java
// Disable the filter.
mEffects.setFilter(null);
```
:::
:::if{props.platform="Flutter"}
```dart
// Disable the filter.
ZegoEffectsPlugin.instance.setFilter('');
```
:::
:::if{props.platform="RN"}
```javascript
// Disable the filter.
mEffects.setFilter(null);
```
:::
