# Cancel an upload task

- - -

## Overview

Call this method to cancel an upload task.

<Note title="Note">

- This method can only cancel tasks that are pending; it is not applicable to tasks that have started or completed.
- To check the cancellation result, please use the [callback](./server%20APIs%20callback.mdx) or the [Describe upload tasks](./Describe%20upload%20tasks.mdx) method.
</Note>

## Request method and endpoint

- Request method: POST
- Request URL: `https://cloud-player-api.zego.im/?Action=CancelUploadTask`
- Protocol: HTTPS
- Rate limit: 20 times/second

&AppId=1234567890
&SignatureNonce=15215528852396
&Timestamp=1234567890
&Signature=7a2c0f11145fb760d607a07b54825013
&SignatureVersion=2.0
