# 取消上传任务

- - -

## 描述

可通过本接口取消一个上传任务。

<Note title="Note">

- 本接口只能取消正在等待中的任务， 不适用于已开始或已结束的任务。
- 取消结果请通过 [回调\|_blank](!cloud_player-server_APIs_callback) 或 [查询上传任务\|_blank](!cloud_player-DescribeUploadTasks) 接口判断。
</Note>

## 接口原型

- 请求方法：POST
- 请求地址：https://cloud-player-api.zego.im/?Action=CancelUploadTask
- 传输协议：HTTPS
- 调用频率限制：20 次/秒

## 请求示例

- 请求 URL：
    ```json
    https://cloud-player-api.zego.im/?Action=CancelUploadTask
    &AppId=1234567890
    &SignatureNonce=15215528852396
    &Timestamp=1234567890
    &Signature=7a2c0f11145fb760d607a07b54825013
    &SignatureVersion=2.0
    &IsTest=false
    ```

- 请求消息体：
    ```json
    {
        "UploadTaskId": "XXXXXXXXXXXX"
    }
    ```
