# Face beautification

- - -

## Feature Overview

The ZegoEffects SDK provides face beautification features that help users achieve natural and attractive skin appearance through unique face beautification effects.

Developers can adjust the intensity of skin tone enhancement, skin smoothing, image sharpening and other effects to achieve the desired beautification results.

## Prerequisites

Before using the ZegoEffects SDK face beautification features, please ensure:

- You have integrated the ZegoEffects SDK and implemented basic image processing functionality. For details, see [Implement basic image processing|_blank](!Quick_Starts/Solution_Implementation).
- You have imported the basic "CommonResources" resource files. For details, see [Import resources and models|_blank](!Quick_Starts/Import_Resources).

## Implementation Steps

Face beautification includes features like skin tone enhancement, skin smoothing, image sharpening, cheek blusher, etc.

After enabling face beautification, developers can customize the beautification parameters based on actual needs. If not customized, the SDK will use default parameter values.

### Skin Tone Enhancement 

<div class = 'mk-warning'>

Before using skin tone enhancement, please import the corresponding "FaceWhiteningResources" resource file, otherwise the feature will not take effect. For details, see [Import resources and models|_blank](!Quick_Starts/Import_Resources).
</div>

Call the [enableWhiten|_blank](@enableWhiten) interface to enable skin tone enhancement, and call [setWhitenParam|_blank](@setWhitenParam) to set the enhancement intensity.

The intensity parameter of [ZegoEffectsWhitenParam|_blank](@-ZegoEffectsWhitenParam) ranges from [0, 100], with a default value of "50". A larger value means stronger enhancement. Developers can customize the value based on needs.

```java
// Enable skin tone enhancement
mEffects.enableWhiten(true);
// Set enhancement intensity, range [0, 100], default is 50
mEffects.setWhitenParam({ intensity: 100 });
```

### Remove Nasolabial Folds

Call the [enableWrinklesRemoving|_blank](@enableWrinklesRemoving) interface to enable the nasolabial fold removal feature, and call the [setWrinklesRemovingParam|_blank](@setWrinklesRemovingParam) interface to set the degree of nasolabial fold removal.

The "intensity" parameter of the [ZegoEffectsWrinklesRemovingParam|_blank](@-ZegoEffectsWrinklesRemovingParam) object, which represents the degree of nasolabial fold removal, has a value range of [0, 100], with a default value of "50". The higher the value, the greater the degree of nasolabial fold removal. Developers can customize the value according to their needs.

```js
// Enable nasolabial fold removal feature
mEffects.enableWrinklesRemoving(true);

// Set the degree of nasolabial fold removal, range [0, 100], default is 50
mEffects.setWrinklesRemovingParam({ intensity: 100 });
```

### Skin Tone Adjustment Effect

<div class = 'mk-warning'>

Before using the skin tone adjustment feature, please import and load the corresponding "SkinColorResources" skin resource files, otherwise the feature will not work properly. For details, please refer to [Import resources and models|_blank](!Quick_Starts/Import_Resources).
</div>

Call the [enableSkinColor|_blank](@enableSkinColor) interface to enable the skin tone adjustment feature, call the [setSkinColorResPath|_blank](@setSkinColorResPath) interface to load the supported skin resources, and then call the [setSkinColorParam|_blank](@setSkinColorParam) interface to set the degree of skin tone adjustment.

The "intensity" parameter of the [ZegoEffectsSkinColorParam|_blank](@-ZegoEffectsSkinColorParam) object, which represents the degree of skin tone adjustment, has a value range of [0, 100], with a default value of "0". The higher the value, the more noticeable the skin color change. Developers can customize the value according to their needs.

```js
// Enable skin tone adjustment feature
mEffects.enableSkinColor(true);

// Load the supported/specified skin tone resources
String skinResPath = "SkinColorResources/***";
mEffects.setSkinColorResPath(skinResPath);

// Set the degree of skin tone adjustment, range [0, 100], default is 0;
mEffects.setSkinColorParam({ intensity: 100 });
```
