## 1 Prerequisites

Please ensure your development environment meets the following requirements:

* React Native 0.73.6 or above.
* iOS 13.0 or above and an iOS device or simulator that supports audio and video (real device recommended).
* Android 8.0 or above and an Android device or simulator that supports audio and video (real device recommended). For real devices, please enable "USB debugging".
* iOS/Android device connected to the Internet.
* [Node.js|_blank](https://nodejs.org/en/) installed, LTS version recommended as shown on their homepage.

## 2 Requirements

Please ensure the following requirements are met before running:
* Registered an account on [ZEGOCLOUD Console](https://console.zegocloud.com/account/login) and obtained the AppID and AppSign required for SDK initialization.
* Contacted ZEGOCLOUD technical support to obtain the License file for beauty effects.

## 3 Run the Sample Code

1. Download the latest version of [iOS ZegoEffects SDK|_blank](#14908) or [Android ZegoEffects SDK|_blank](#14909), extract it, and import the `Resources` and `Models` folders into your downloaded [sample code|_blank](https://github.com/ZEGOCLOUD/zego_effects_reactnative_demo).

2. Add a `KeyCenter.js` file in the root directory of the sample code with the following content. Fill in your `appID`, `appSign`, and `effectLicense`. For how to obtain these, please refer to [Prerequisites|_blank](18045#2).

    ```js
    const KeyCenter = { 
        // Get appID from https://console.zegocloud.com/account/login
        // For example: const appID = 123456789;
        appID: ,
        // Get appSign from https://console.zegocloud.com/account/login, for example: "39011cbxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"
        appSign: '',
        effectLicense: '',
    }

    export default KeyCenter
    ```

4. In the root directory, install dependencies and run the sample code.

    ```bash
    yarn

    # Run iOS
    yarn ios

    # Run Android
    yarn android
    ```
