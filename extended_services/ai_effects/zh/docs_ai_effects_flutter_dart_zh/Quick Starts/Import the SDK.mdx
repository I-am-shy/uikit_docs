# 集成 SDK 

--- 

## 1 准备环境  

在开始集成 ZegoEffects SDK 前，请确保开发环境的 Flutter 为 2.0 或以上版本

## 2 集成 SDK  


1. 打开 “pubspec.yaml” 文件，添加 “zego_effects_plugin” 依赖：
    ```
    zego_effects_plugin: ^2.1.0
    ``` 

2. 项目根目录下执行如下命令，导入 SDK。
    ```
     flutter pub get
    ```
