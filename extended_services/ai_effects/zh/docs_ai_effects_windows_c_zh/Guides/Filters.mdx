# 滤镜

- - -

## 1 功能简介

ZegoEffects SDK 提供滤镜功能，包含多种滤镜风格可使用，可广泛应用于短视频，直播等场景。

## 2 前提条件

在使用 ZegoEffects SDK 滤镜功能前，请确保已在项目中集成 ZegoEffects SDK，实现基本的图像处理功能，详情请参考 [快速开始 - 集成 SDK\|_blank](!Quick_Starts/Integration)。



## 3 使用步骤

滤镜功能提供了以下风格：
- 自然（Natural）：奶油（Creamy）、青春（Brighten）、清新（Fresh）、秋天（Autumn）
- 灰调（Gray）：莫奈（Cool）、暗夜（Night）、胶片（Film-like）
- 梦境（Dreamy）：落日（Sunset）、琉璃（Cozily）、星云（Sweet）

开发者在调用接口时，可以根据需要，选择对应的滤镜资源使用。

开启滤镜功能之后，开发者可根据实际的需求来自定义滤镜参数。如果未自定义，SDK 将采用默认参数值实现滤镜功能。

### 3.1 滤镜

1. 开启滤镜。

    调用 [zego_effects_set_filter\|_blank](@zego_effects_set_filter) 接口开启滤镜功能，传入对应的滤镜绝对路径。调用 [zego_effects_set_filter_param\|_blank](@zego_effects_set_filter_param) 接口设置滤镜强度。

    [zego_effects_filter_param\|_blank](@-zego_effects_filter_param) 对象的滤镜强度参数 “intensity” 的取值范围为 [0, 100]。取值越大，效果呈现的程度越大，开发者可以根据需求自定义取值。

    <div class = 'mk-warning'>

    滤镜风格的效果不能叠加，设置新的滤镜路径后，会覆盖之前的滤镜效果。
    </div>

    ```c
    // 开启滤镜功能
    // 1. 开启滤镜功能，并且传入对应的滤镜绝对路径
    zego_effects_set_filter(handle,"/xxx/xxx/Night.bundle");

    // 设置滤镜强度，范围 [0, 100]
    zego_effects_filter_param param;
    param.intensity = 100;
    zego_effects_set_filter_param(handle,param);
    ```

2. 移除滤镜。

    如果需要移除滤镜，可将滤镜资源路径设置为空，传入 [zego_effects_set_filter\|_blank](@zego_effects_set_filter) 接口。

    ```c
    // 移除滤镜功能
    zego_effects_set_filter(handle,"");
    ```
