# 在锁屏时显示来电呼叫


本文档描述了如何配置在锁屏时显示来电呼叫。


## 实现流程

为了在锁屏时显示来电呼叫，请在你的app模块的manifest文件中添加以下权限即可：
```xml
<uses-permission android:name="android.permission.USE_FULL_SCREEN_INTENT" />
```

注意：
1. 如果你的应用需要上传到 google play store，由于[google play store 的政策](https://support.google.com/googleplay/android-developer/answer/13392821?hl=en),你可能需要针对这两个权限作出相应的声明才可以避免被拒绝。

2. 如果你的设备有针对锁屏显示的额外的设置，允许它们后此功能才能正常工作。
